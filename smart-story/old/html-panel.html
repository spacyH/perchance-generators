📥 Import
Paste, upload, or link to a character / chat from FurAI, ai-character-chat, TavernAI, or a Smart Story backup. The format is detected automatically.
Supported: .json backups, .jsonl SillyTavern group chat exports, FurAI exports, TavernAI cards, plain text character files.
Accepts: ai-character-chat share links (?data=NAME~FILEID), FurAI character links (?char_import=...), or any direct URL to a JSON file.
Format
—
—
Import & Continue: full import into a new scene, including any message history.
Use as Current: replace your currently-active character with this one (your library keeps it too). Messages are discarded.
Add to Library: save the character into your library for use in future scenes. Doesn't touch your current character or chat. Use this to build a multi-character library by importing cards one at a time.
📤 Export
Choose a format. Smart Story backups are best for moving between devices. TavernAI cards are character-only — they work in SillyTavern, RisuAI, Janitor, and most other AI chat apps. ai-character-chat exports keep chat history along with the character.
Includes every chat, character, persona, settings, and avatars in this app.
📖 Who's in this scene?
Pick which characters appear in this scene. The first one ticked is the primary speaker (opens the scene). When 2 or more are picked, a "Reply as:" dropdown will appear in the composer so you can switch who answers your messages.
✚ Bring in a character
Pick a character from your library to enter this scene. A short system note will be added to the conversation so the AI knows the new character has arrived. To create a new character first, open Settings → CHARACTER.
