<h1>[$output(1, true)] Pride Plugin [$output(1, true)]</h1>

<div style="margin:0 auto;width:100%;max-width:800px;background:white;border-radius:2px;padding:1em; box-sizing:border-box;">
	<p style="text-align:justify; margin-top:0;">This is a very simple little plugin that automatically shows a rainbow flag image in June (pride month!), and hides it for the rest of the months. You can also display other flags/text on other months/days, as explained further down this page.</p>
  <p>To start with, write this in your Perchance code panel:</p>
	 
<pre class="custom" style="margin:0; height:100%; box-sizing:border-box;">
pride = \{import:pride-plugin\}
</pre>
	
  <p>and then write this in your HTML (bottom-right) panel:</p>
	
<pre class="custom" style="margin:0; height:100%; box-sizing:border-box;">
\[pride()\]
</pre>

	<p>and it'll output a flag like this if it's June:</p>
	
	<p>[$output(1, true)]</p>
	
	<p>You can change the size of the flag like this:</p>
	
<pre class="custom" style="margin:0; height:100%; box-sizing:border-box;">
\[pride(2)\]
</pre>
	
	<p>That outputs a flag that is 2x the default size:</p>
	
	<p>[$output(2, true)]</p>
	
	<p>Just change the <code>2</code> to any desired size (decimal numbers work too!).</p>
	<p>Want the flag to be a link to a website? Here's an example of how to do that:</p>
	
<pre class="custom" style="margin:0; height:100%; box-sizing:border-box;">
&lt;a target="_blank" href="https://en.wikipedia.org/wiki/Gay_pride"&gt;\[pride(2)\]&lt;/a&gt;
</pre>
	
	<p><a target="_blank" href="https://en.wikipedia.org/wiki/Gay_pride">[$output(2, true)]</a></p>
	
	<p>So now if you click the flag, it'll take you to the URL between the quotes :)</p>
	
	<p>If you want it to show all year round (not just in June), use this:</p>
	
<pre class="custom" style="margin:0; height:100%; box-sizing:border-box;">
\[pride(1, true)\] <span style="opacity:0.5">// default size, all year round</span>
\[pride(2, true)\] <span style="opacity:0.5">// twice the size, all year round</span>
<span style="opacity:0.5">// etc.</span>
</pre>
 <p>You can also display your own custom flags/text/HTML on specific months, and specific days of specific months like this:</p>  
<pre class="custom" style="margin:0; height:100%; box-sizing:border-box;">
dateText
  june = 🌈
  march 31 = 🏳️‍⚧️
  january = ☾⋆｡𖦹 °✩ 
  december 25 = 🎄🎁
  default = :) <span style="opacity:0.5">// this gets displayed if there are no matches</span>

output
  \[pride(dateText)\]
</pre>
	<p>You can remove the <code>default = ...</code> line if you don't want to display anything on non-matching days/months.</p>  
	<br><br>
	<p><b>Notes:</b></p>
	<ul class="dot-list">
		<li>I wanted to use the rainbow flag emoji, but it doesn't show properly in my web browser, and I figured a small image would work better because it's more consistently rendered across devices anyway.</li>

	</ul>
