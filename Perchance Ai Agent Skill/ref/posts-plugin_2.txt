This is a fork/remix of the <a href="/text-to-image-plugin" target="_blank">posts-plugin</a>, but unfortunately it's a really bad idea to fork this particular plugin, since its code is 'coupled' with the server code, so if I need to change the server code, your version of this plugin will likely break. If you'd like new features, best to ask for them on the community forum so your generators don't randomly break in the future when I update the server code. Alternatively, you can create a new plugin that <u>imports this plugin</u> - basically create a "wrapper" plugin that changes/expands on this plugin's behavior.</div>

<h1>Posts Plugin (WIP - not working yet)</h1>

<main>
  <p>This plugin allows you to do stuff like create your own comments plugin, or create a site that's like Lemmy/Facebook/Tumblr/Messenger/etc. You basically have access to a full 'database' that you can add stuff to, with support for voting, sorting by newest, or score, adding and filtering by tags, etc.</p>  
  <p>To use this plugin, you'll first need to import it by adding this line to your lists editor:</p>
<pre>
posts = \{import:posts-plugin\}
</pre>
  <p>And now try putting this in your lists editor:</p>
<pre>
article
  title = \[titleEl.value\]
  content = \[contentEl.value\]
  foo = 123
</pre> 
  <p>And put this in the HTML panel:</p>
<pre>
&lt;input id="titleEl"&gt;
&lt;textarea id="contentEl">&lt;/textarea&gt;
&lt;button onclick="posts.blog.add(article)"&gt;submit&lt;/button&gt;
&lt;div&gt;\[$once:posts.blog.feed(\{sort:"trending", period:"week"\})\]&lt;/div&gt;
</pre> 
  <p><a href="https://perchance.org/posts-plugin-example-1#edit" target="_blank">Here's an example generator</a> that uses the above code.</p>
