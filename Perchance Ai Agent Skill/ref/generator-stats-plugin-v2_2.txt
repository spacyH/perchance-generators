<h1>Generator Stats Plugin (UPDATED!)</h1>
<div style="margin:0 auto;width:100%;max-width:780px;background:pink;border-radius:30px;padding:1em;box-sizing:border-box;border:5px groove purple;filter:drop-shadow(10px 10px 5px hotpink)">
<h3>It will no longer say simply '1k' or '5.7k', it'll say '1,033' or '5,726' instead! Or '2,678' or '6,609' or '98,001' or '8,211'! This is what ya'll have been waiting for! Here it is!<br>Oh, btw- Raggedflight gave me an idea. I think it's pretty good-</h3>
[com()]
	<p>Put this in your Perchance code panel:</p>
	<pre class="custom">generatorStats = \{import:generator-stats-plugin-v2\}</pre>

	<p>And now, whenever you write this:</p>
	<pre class="custom">\[generatorStats("views")\]</pre>
	<p>It will be replaced with the number of people who have viewed your generator. You can also use this to output how long ago it was since you edited the generator:</p>
	<pre class="custom">\[generatorStats("lastEditTime")\]</pre>
	
	<p><b>Advanced Usage:</b> If you want to load the stats of a *different* generator (not necessarily your own generator), then you need to give the name of the generator <b>in quotation marks</b> like in these examples where we get the number of views for a few different plugins:</p>
<pre class="custom">
\[generatorStats("generator-stats-plugin", "views")\] <span style="opacity:0.3;">= [$output("generator-stats-plugin", "views")]</span>
\[generatorStats("background-image-plugin", "views")\] <span style="opacity:0.3;">= [$output("background-image-plugin", "views")]</span>
\[generatorStats("dice-plugin", "views")\] <span style="opacity:0.3;">= [$output("dice-plugin", "views")]</span>
</pre>
	<p>This could be handy if, for example, you made a "profile page" to list all your generators, and you wanted to display each of your generators along with the number of views that each one has. Here's a <a href="https://perchance.org/very-simple-profile-page-example#edit" target="_blank">simple profile page example</a> that uses this plugin to show stats for each generator.</p>
