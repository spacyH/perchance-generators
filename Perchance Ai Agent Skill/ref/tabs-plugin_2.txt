
<h1>Tabs Plugin</h1>

<div style="margin:0 auto;width:100%;max-width:800px;background:white;border-radius:2px;padding:1em; box-sizing:border-box;">
	<p style="text-align:justify; margin-top:0;">This plugin lets you insert a "tab viewer" thing where each tab can have randomly generated content as specified by you. Simply put this in your perchance code panel:</p>

<pre class="custom">
tabs = \{import:tabs-plugin\}

tabList
  *
	  title = Title 1
		content = My \{neat|cool\} content 1
  *
	  title = Title 2
		content = My \{awesome|great\} content 2
  *
	  title = Title 3
		content = My \{nice|lovely\} content 3
</pre>
	
	<p>And then put this somewhere in your HTML panel (bottom-right panel):</p>
	
<pre class="custom">
\[tabs(tabList)\]
</pre>
	
	<p>And you'll get this (try clicking another tab):</p>
	
	[$output(tabList)]
	
	<p>You can put as many tabs as you like - just follow the format of the example code above.</p>
	
	<p>You can specify a default tab to display first like this:</p>

<pre class="custom">
  *
	  title = Title 3
		content = My \{awesome|great\} content 3
		default = true
</pre>
	
	<p>If you don't specify a default tab, the first tab will be shown first as you can see in the above example.</p>

	<p>If you want it to "remember" the active/focussed tab and stay on that one even when the user clicks "randomize", then you can do it like this:</p>
	
<pre class="custom">
tabList
  rememberActiveTab = true
  *
	  ...
	*
	  ...
</pre>
  
  <p>What if you want a "randomize" button to only update a specific tab? In that case you just need to give the tab an <code>id</code> (e.g. <code>myCoolTab</code> in the example below), and then if you run <code>update(myCoolTab)</code> when a button is clicked, it'll only update that tab. In this example we put the button <i>inside</i> the tab:</p>
<pre class="custom">
  *
	  title = Title 3
		content = My \{awesome|great\} content 3 &lt;button onclick="update(myCoolTab)"&gt;randomize&lt;/button&gt;
		id = myCoolTab
</pre>
  <p><a href="https://perchance.org/update-individual-tabs-plugin-example#edit" target="_blank">Here's an example generator</a> which does that.</p>
  
  <p>You can also change the background color, but you need to provide the color as a <a href="https://www.google.com/search?q=hex+color+picker" target="_blank">hex color code</a> like <code>#a23b1a</code> (a hash, followed by 6 letters/numbers), and <b>if you make edits to the background color you you need to save your generator and refresh the whole page</b> to see your changes.</p>
<pre class="custom">
tabList
  backgroundColor = #000000
  ...
</pre>
  <p>You can also set <code>borderColor</code> and <code>textColor</code> in the same way. Remember to refresh to see changes for those too.</p>
	<br><br>
	<p><b>Notes:</b></p>
	<ul class="dot-list">
		<li>Check out <a href="https://perchance.org/simple-tabs-plugin-example#edit" target="_blank">this simple example</a>.</li>
    <li>And <a href="https://perchance.org/tabs-plugin-example-1#edit" target="_blank">here's an example</a> with some more complex tab content (images, etc.).</li>
		<li>Note that if you put too many tabs in, it won't display very nicely for people with narrow screens (e.g. on mobile phones) because the tabs get all squashed up and start stacking on top of one another.</li>
		<li>Also check out <a href="https://perchance.org/power-tabs-plugin" target="_blank">power-tabs-plugin</a>, a community-made plugin which has more features.</li>
		<li>Check out more plugins at <a href="/plugins">perchance.org/plugins</a></li>
	</ul>
</div>
