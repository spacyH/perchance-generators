<h1>Markdown Plugin</h1>

<div style="margin:0 auto;width:100%;max-width:750px;background:white;border-radius:2px;padding:1em; box-sizing:border-box;">
	<p style="text-align:justify; margin-top:0;">Markdown is sort of like a more simple version of HTML. Have a look at <a href="https://dillinger.io">this</a> to get a idea of what it is. As a small example, to make bold text in HTML, you write &lt;b&gt;hello&lt;b&gt; whereas in markdown you simply write **hello**. This plugin converts the simple markdown syntax (that is made for human-readability) into HTML code that the browser understands. That's a very simple example, but you can do all sorts of things in markdown - even tables, numbered lists, and more (see that previous link for examples). <a href="/markdown-plugin-example#edit">Here's an example generator</a> that uses this plugin.</p>
	<p style="text-align:justify;">Using the plugin is simple - just copy and paste this in your Perchance code panel, and then customize the <code>text</code> list:</p>
<pre class="custom">
markdown = \{import:markdown-plugin\}

output = \[markdown(text)\]

text
  # My title
	The first line of my first paragraph.
	Second line of first paragraph.
	\\s <span style="opacity:0.4">// &lt;-- paragraph break</span>
	You can *italicize* or **bold** your text easily.
	
	## Sub-header
	Here's a list of items:
	  * Bullet point 1
		* Bullet point 2
		* Bullet point 3
		  * Sub-bullet-point
		* Bullet point 4
	\\s  <span style="opacity:0.4">// &lt;-- end lists like this too</span>
	And we can do numbered lists too:
	  1. First item
		2. Second item
		3. ~~Scratch this~~
	\\s
	Adding a \[link\](https://dillinger.io) is easy with markdown.
	Images are easy too:
	\\s
	!\[image hover text\](https://web.archive.org/web/20091018220201/http://www.geocities.com/roqofages/Rainbowflagwaving.gif)
	\\s
	And you can still put HTML code in if needed:
	\\s
	&lt;video controls src\\="https://i.imgur.com/26DZgXM.mp4"&gt;&lt;/a&gt;
	```
	You can do code-blcks like this.
	Second line of code.
	  Third line of code (indented)
	```
	> And quoted text like this. Quotes automatically wrap: foo bar foo bar foo bar foo bar foo bar foo bar foo bar foo bar foo bar foo bar foo bar foo bar foo bar foo bar foo bar foo bar foo bar foo bar foo bar foo bar.
	\\s
	> Break quotes up into paragraphs using the \\s trick.
	\s
	Header  | Header
	--------|---------
	You     | Can
	Make    | Neat
	Little  | Tables
	Too     | !
	\s
	So that's a brief intro to some of markdown's basic features!
</pre>
	
	<p>And now writing <code>\[output\]</code> in the HTML panel will produce this:</p>
	
	[$output(text)]
