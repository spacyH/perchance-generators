      <h1>weld.undo</h1>
      <span class="ver">v1.0.0</span>
    </div>
    <p class="lead">Undo and redo with state snapshots, atomic transactions, and keyboard shortcuts. Works with any state shape.</p>
  </div>
  <h2>Overview</h2>
<p>weld.undo gives you undo and redo for any application state, without any framework. You define how to read and write your state, and the plugin manages a stack of snapshots. Transactions let you group multiple changes into one atomic undo step; an optional keyboard binding installs Cmd+Z and Cmd+Shift+Z (and Cmd+Y) handlers.</p>
<p>Snapshots are deep clones -- the plugin uses <code>structuredClone</code> where available and falls back to JSON round-trip. That means your state needs to be serializable, but you don't need to write any merge/diff logic.</p>

<h2>Core API</h2><h3>weld.undo.create(\{getState, setState, maxDepth?\})</h3>
<div class="sig" data-codeid="sig3"></div>
<p>Create a new stack. Both <code>getState</code> and <code>setState</code> are required.</p>
<ul>
<li><code>getState()</code> returns the current state -- must be serializable.</li>
<li><code>setState(state)</code> applies a state. The plugin passes you a deep clone, so you can mutate freely without affecting history.</li>
<li><code>maxDepth</code> caps the stack size (default 50). Oldest snapshots drop off when exceeded.</li>
</ul>
<h3>stack.snapshot(label?)</h3>
<div class="sig" data-codeid="sig4"></div>
<p>Take a snapshot of the current state and push it onto the stack. The optional <code>label</code> is for your own debugging (shown in <code>history()</code>). Returns the new snapshot, or <code>null</code> if called inside a transaction.</p>
<pre data-codeid="code5"></pre>

<p>Taking a snapshot truncates any redo history past the current position -- this matches editor expectations (after undo, doing something new clears the future).</p>
<h3>stack.undo() / stack.redo()</h3>
<div class="sig" data-codeid="sig6"></div>
<div class="sig" data-codeid="sig7"></div>
<p>Move backward or forward in history. Calls <code>setState</code> with the appropriate snapshot. Return the snapshot that was applied, or <code>null</code> if at the boundary.</p>
<h3>stack.canUndo() / stack.canRedo()</h3>
<p>Booleans -- useful for enabling/disabling toolbar buttons.</p>
<pre data-codeid="code8"></pre>
<h2>Transactions</h2><h3>stack.transaction(fn, label?)</h3>
<div class="sig" data-codeid="sig9"></div>
<p>Run multiple state changes as one atomic undo step. Snapshots taken <em>inside</em> the function are deferred -- only one snapshot is created when the function returns. Supports both sync and async functions.</p>

<pre data-codeid="code10"></pre>

<p>Async transactions work too:</p>
<pre data-codeid="code11"></pre>

<p>If the function throws (or the Promise rejects), no snapshot is taken -- the state stays whatever you left it as. Roll back manually if you need to.</p>
<h2>Inspection</h2><h3>stack.peek() / .depth() / .position()</h3>
<p><code>peek()</code> returns the current snapshot. <code>depth()</code> is the total number of snapshots. <code>position()</code> is the cursor's index (0 = oldest).</p>
<pre data-codeid="code12"></pre>
<h3>stack.history()</h3>
<div class="sig" data-codeid="sig13"></div>
<p>Returns a copy of the entire snapshot array, oldest first. Useful for building a history panel or timeline UI.</p>
<pre data-codeid="code14"></pre>
<h3>stack.clear()</h3>
<p>Drop all snapshots. The current state is unchanged, but undo/redo no longer works until you take a new snapshot.</p>
<h2>Reactive UI</h2><h3>stack.onChange(fn)</h3>
<div class="sig" data-codeid="sig15"></div>
<p>Get called every time the stack changes. <code>event</code> is one of <code>'snapshot'</code>, <code>'undo'</code>, <code>'redo'</code>, or <code>'clear'</code>. Use this to update toolbar button state, status bars, or live history panels.</p>
<pre data-codeid="code16"></pre>
<h2>Keyboard shortcuts</h2><h3>stack.bindKeyboard(\{target?, preventDefault?\})</h3>
<div class="sig" data-codeid="sig17"></div>
<p>Install global keyboard handlers for Cmd+Z (undo), Cmd+Shift+Z (redo), and Cmd+Y (redo). Ctrl+ also works on Windows/Linux. Returns an unbind function.</p>
<ul>
<li><code>target</code>: where to attach the listener. Defaults to <code>document</code>.</li>
<li><code>preventDefault</code>: whether to call <code>preventDefault()</code> on matched keys. Defaults to true.</li>
</ul>
<pre data-codeid="code18"></pre>

<p>Only one binding is installed per stack -- calling <code>bindKeyboard</code> twice returns the same unbind function.</p>
<h2>Common patterns</h2><h3>A toolbar with undo/redo buttons</h3>
<pre data-codeid="code19"></pre>
<h3>Snapshotting at coarse boundaries, not every keystroke</h3>
<p>Snapshotting on every keystroke makes undo unusable (one undo per character). Snapshot on logical events instead -- when the user finishes typing, or commits a change.</p>
<pre data-codeid="code20"></pre>
<h2>Caveats</h2><div class="warn">
<strong>Serializable state only.</strong> Snapshots are JSON round-trips by default (with <code>structuredClone</code> upgraded if available). That means DOM elements, functions, class instances with prototypes, and circular references won't survive. Keep your state plain objects, arrays, primitives, dates, and the like.

<script>
(function(){
  'use strict';

  if (!window.weld) window.weld = {};
  if (window.weld.undo) return; // already loaded

  var stacks = [];

  // -----------------------------------------------------------------
  // DEEP CLONE
  // -----------------------------------------------------------------
  function deepClone(v) {
    if (v == null || typeof v !== 'object') return v;
    if (typeof structuredClone === 'function') {
      try { return structuredClone(v); } catch (e) { /* fall through */ }
    }
    try { return JSON.parse(JSON.stringify(v)); }
    catch (e) { throw new Error('weld.undo: state is not serializable'); }
  }

  // -----------------------------------------------------------------
  // STACK FACTORY
  // -----------------------------------------------------------------
  function createStack(opts) {
    opts = opts || {};
    var maxDepth = opts.maxDepth || 50;
    var getState = opts.getState;
    var setState = opts.setState;

    if (typeof getState !== 'function' || typeof setState !== 'function') {
      throw new Error('weld.undo.create requires opts.getState and opts.setState functions');
    }

    var snapshots = [];   // each: state / label / timestamp
    var cursor = -1;
    var inTransaction = false;
    var transactionLabel = null;
    var listeners = [];
    var bound = false;
    var unbindKbd = null;

    function notify(event) {
      for (var i = 0; i < listeners.length; i++) {
        try { listeners[i](event, api); } catch (e) { /* swallow */ }
      }
    }

    function snapshot(label) {
      if (inTransaction) return null;  // deferred
      var s = {
        state: deepClone(getState()),
        label: label || '',
        timestamp: Date.now()
      };
      // truncate any redo history past cursor
      if (cursor < snapshots.length - 1) {
        snapshots = snapshots.slice(0, cursor + 1);
      }
      snapshots.push(s);
      while (snapshots.length > maxDepth) snapshots.shift();
      cursor = snapshots.length - 1;
      notify('snapshot');
      return s;
    }

    function undo() {
      if (!canUndo()) return null;
      cursor--;
      var s = snapshots[cursor];
      try { setState(deepClone(s.state)); }
      catch (e) { cursor++; throw e; }
      notify('undo');
      return s;
    }

    function redo() {
      if (!canRedo()) return null;
      cursor++;
      var s = snapshots[cursor];
      try { setState(deepClone(s.state)); }
      catch (e) { cursor--; throw e; }
      notify('redo');
      return s;
    }

    function canUndo() { return cursor > 0; }
    function canRedo() { return cursor >= 0 && cursor < snapshots.length - 1; }

    function transaction(fn, label) {
      if (typeof fn !== 'function') throw new Error('transaction requires a function');
      if (inTransaction) return fn();   // nested: just run

      inTransaction = true;
      transactionLabel = label || '';

      var finish = function() {
        inTransaction = false;
        snapshot(transactionLabel);
      };

      try {
        var result = fn();
        if (result && typeof result.then === 'function') {
          // async: resolve, then snapshot
          return result.then(
            function(r) { finish(); return r; },
            function(err) { inTransaction = false; throw err; }
          );
        }
        finish();
        return result;
      } catch (e) {
        inTransaction = false;
        throw e;
      }
    }

    function clear() {
      snapshots.length = 0;
      cursor = -1;
      notify('clear');
    }

    function peek() { return cursor >= 0 ? snapshots[cursor] : null; }
    function history() { return snapshots.slice(); }
    function depth() { return snapshots.length; }
    function position() { return cursor; }

    function onChange(fn) {
      if (typeof fn !== 'function') throw new Error('onChange requires a function');
      listeners.push(fn);
      return function() {
        var i = listeners.indexOf(fn);
        if (i !== -1) listeners.splice(i, 1);
      };
    }

    function bindKeyboard(kbdOpts) {
      if (bound) return unbindKbd;
      kbdOpts = kbdOpts || {};
      var target = kbdOpts.target || document;
      var preventDefault = kbdOpts.preventDefault !== false;

      function handler(e) {
        var meta = e.metaKey || e.ctrlKey;
        if (!meta) return;
        var k = e.key.toLowerCase();
        if (k === 'z') {
          if (e.shiftKey) {
            if (canRedo()) { if (preventDefault) e.preventDefault(); redo(); }
          } else {
            if (canUndo()) { if (preventDefault) e.preventDefault(); undo(); }
          }
        } else if (k === 'y') {
          if (canRedo()) { if (preventDefault) e.preventDefault(); redo(); }
        }
      }
      target.addEventListener('keydown', handler);
      bound = true;
      unbindKbd = function() {
        target.removeEventListener('keydown', handler);
        bound = false;
        unbindKbd = null;
      };
      return unbindKbd;
    }

    // Take initial snapshot so the user has a baseline
    snapshot('initial');

    var api = {
      snapshot: snapshot,
      undo: undo,
      redo: redo,
      canUndo: canUndo,
      canRedo: canRedo,
      transaction: transaction,
      clear: clear,
      peek: peek,
      history: history,
      depth: depth,
      position: position,
      onChange: onChange,
      bindKeyboard: bindKeyboard,
      maxDepth: maxDepth
    };

    stacks.push(api);
    return api;
  }

  // -----------------------------------------------------------------
  // API
  // -----------------------------------------------------------------
  window.weld.undo = {
    create: createStack,
    stacks: function() { return stacks.slice(); },
    version: '1.0.0'
  };
})();

</script>
