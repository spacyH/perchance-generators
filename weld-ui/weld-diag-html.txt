      <h1>weld.diag</h1>
      <span class="ver">v1.0.0</span>
    </div>
    <p class="lead">Ring-buffer diagnostic logger with auto-capture and shareable support bundles. Part of the weld plugin suite.</p>
  </div>
  <h2>Overview</h2>
<p>weld.diag is a small ring-buffer logger that captures errors, warnings, and arbitrary events from your generator. It hooks <code>console.error</code>, <code>console.warn</code>, <code>window.onerror</code>, and <code>unhandledrejection</code> automatically -- and bundles everything (with browser info and detected plugin versions) into a copyable or downloadable JSON blob.</p>
<p>The intended use is bug reports. When a user hits a problem, they click a "copy diagnostic bundle" button, paste it into a GitHub issue or Discord message, and you get every recent error plus enough environment context to reproduce. No telemetry, no network calls, no third party -- all in memory, user controls every byte.</p>

<h2>The entry format</h2>
<p>Every log entry has the same shape:</p>
<pre data-codeid="code3"></pre>

<p>The <code>source</code> field tells you where the entry came from. <code>manual</code> means you logged it yourself. <code>console</code> means it came from console.error or console.warn. <code>window</code> is a window.onerror handler. <code>promise</code> is an unhandled rejection.</p>
<h2>Manual logging</h2><h3>weld.diag.log(message, data?)</h3>
<div class="sig" data-codeid="sig4"></div>
<p>Log an info-level entry from manual code. <code>data</code> is optional and can be anything JSON-serializable.</p>
<pre data-codeid="code5"></pre>
<h3>weld.diag.info / .warn / .error / .debug</h3>
<p>Same as <code>.log</code> but with a different <code>level</code>. <code>error</code> and <code>warn</code> are what auto-capture writes; <code>info</code> and <code>debug</code> are for your own breadcrumbs.</p>
<pre data-codeid="code6"></pre>
<h3>weld.diag.record(level, source, message, data?)</h3>
<div class="sig" data-codeid="sig7"></div>
<p>Lower-level form that lets you set a custom <code>source</code>. Useful when you want a separate channel like <code>'api'</code> or <code>'auth'</code> for filtering later.</p>
<pre data-codeid="code8"></pre>
<h2>Querying</h2><h3>weld.diag.entries() / .count()</h3>
<div class="sig" data-codeid="sig9"></div>
<p>Get a snapshot of every entry in the buffer. Returns a copy, not the live array.</p>
<pre data-codeid="code10"></pre>
<h3>weld.diag.filter(predicate)</h3>
<div class="sig" data-codeid="sig11"></div>
<p>If you pass a string, filter by level. If you pass a function, run it against each entry. Returns matching entries.</p>
<pre data-codeid="code12"></pre>
<h3>weld.diag.clear()</h3>
<div class="sig" data-codeid="sig13"></div>
<p>Wipe the buffer. Returns how many entries were dropped.</p>
<h2>Support bundles</h2><h3>weld.diag.bundle(opts?)</h3>
<div class="sig" data-codeid="sig14"></div>
<p>Build a complete diagnostic bundle: every log entry, plus browser info (user agent, platform, viewport), the page URL, and detected versions of all loaded weld plugins. Pass <code>extra</code> to attach arbitrary metadata.</p>
<pre data-codeid="code15"></pre>
<h3>weld.diag.copyBundle(opts?)</h3>
<div class="sig" data-codeid="sig16"></div>
<p>Build a bundle and copy it to the clipboard as pretty-printed JSON. Resolves with the byte size copied. Falls back to a hidden <code>&lt;textarea&gt;</code> + <code>execCommand</code> on browsers without the modern Clipboard API.</p>
<pre data-codeid="code17"></pre>
<h3>weld.diag.downloadBundle(opts?)</h3>
<div class="sig" data-codeid="sig18"></div>
<p>Trigger a browser download of the bundle as a .json file. Same opts as bundle() plus an optional <code>filename</code>.</p>
<pre data-codeid="code19"></pre>
<h2>Live subscriptions</h2><h3>weld.diag.subscribe(fn)</h3>
<div class="sig" data-codeid="sig20"></div>
<p>Get called every time a new entry lands -- useful for live error indicators in your UI. Returns an unsubscribe function.</p>
<pre data-codeid="code21"></pre>
<h2>Configuration</h2><h3>weld.diag.setMaxEntries(n) / .getMaxEntries()</h3>
<p>The buffer defaults to 200 entries. Bump it up if you log heavily, or down if memory matters. Shrinking the limit immediately trims the oldest entries.</p>

<h2>Privacy</h2><div class="warn">
<strong>Privacy note:</strong> bundles can contain anything you logged, including potentially user data. Nothing leaves the page automatically -- it only goes wherever the user pastes or uploads it. Make sure your manual log messages don't include things you wouldn't want a user to share, and consider scrubbing the bundle in your "report a bug" button if needed.

<script>
(function(){
  'use strict';

  if (!window.weld) window.weld = {};
  if (window.weld.diag) return; // already loaded

  // -----------------------------------------------------------------
  // STATE
  // -----------------------------------------------------------------
  var MAX_ENTRIES = 200;
  var entries = [];
  var subscribers = [];
  var hooksInstalled = false;

  // -----------------------------------------------------------------
  // CORE LOGGING
  // -----------------------------------------------------------------
  function nowIso() { return new Date().toISOString(); }

  function safeStr(v) {
    if (v == null) return String(v);
    if (typeof v === 'string') return v;
    if (typeof v === 'number' || typeof v === 'boolean') return String(v);
    if (v instanceof Error) return v.message + (v.stack ? '\n' + v.stack : '');
    try { return JSON.stringify(v); } catch (e) { return '\[unserializable\]'; }
  }

  function record(level, source, message, data) {
    var entry = {
      timestamp: nowIso(),
      level: level,    // debug | info | warn | error
      source: source,  // manual | console | window | promise | <custom>
      message: String(message),
      data: data == null ? null : data
    };
    entries.push(entry);
    while (entries.length > MAX_ENTRIES) entries.shift();
    for (var i = 0; i < subscribers.length; i++) {
      try { subscribers[i](entry); } catch (e) { /* swallow */ }
    }
    return entry;
  }

  // -----------------------------------------------------------------
  // AUTO-CAPTURE HOOKS
  // -----------------------------------------------------------------
  function installHooks() {
    if (hooksInstalled) return;
    hooksInstalled = true;

    // Wrap console.error / console.warn (preserve originals)
    if (console && typeof console.error === 'function' && !console.error.__weldHooked) {
      var origError = console.error.bind(console);
      var hookedError = function() {
        var args = Array.prototype.slice.call(arguments);
        record('error', 'console', args.map(safeStr).join(' '), null);
        return origError.apply(console, arguments);
      };
      hookedError.__weldHooked = true;
      console.error = hookedError;
    }
    if (console && typeof console.warn === 'function' && !console.warn.__weldHooked) {
      var origWarn = console.warn.bind(console);
      var hookedWarn = function() {
        var args = Array.prototype.slice.call(arguments);
        record('warn', 'console', args.map(safeStr).join(' '), null);
        return origWarn.apply(console, arguments);
      };
      hookedWarn.__weldHooked = true;
      console.warn = hookedWarn;
    }

    // window.onerror
    window.addEventListener('error', function(e) {
      record('error', 'window', e.message || 'unknown error', {
        filename: e.filename,
        line: e.lineno,
        col: e.colno,
        stack: e.error && e.error.stack
      });
    });

    // unhandled promise rejections
    window.addEventListener('unhandledrejection', function(e) {
      var reason = e.reason;
      var msg = (reason && reason.message) || safeStr(reason);
      record('error', 'promise', 'Unhandled rejection: ' + msg, {
        stack: reason && reason.stack
      });
    });
  }

  // -----------------------------------------------------------------
  // BUNDLE GENERATION
  // -----------------------------------------------------------------
  function detectPluginVersions() {
    var versions = {};
    if (!window.weld) return versions;
    var names = ['ui', 'migrate', 'backup', 'diag', 'quota', 'undo', 'settings', 'toast', 'markdown', 'chatui', 'lorebook', 'branching'];
    for (var i = 0; i < names.length; i++) {
      var n = names[i];
      if (window.weld[n] && window.weld[n].version) versions[n] = window.weld[n].version;
    }
    return versions;
  }

  function bundle(opts) {
    opts = opts || {};
    var b = {
      generatedAt: nowIso(),
      generator: opts.generator || (typeof generatorName !== 'undefined' ? generatorName : 'unknown'),
      url: location.href,
      userAgent: navigator.userAgent,
      platform: navigator.platform,
      language: navigator.language,
      viewport: { w: window.innerWidth, h: window.innerHeight },
      pluginVersions: detectPluginVersions(),
      entryCount: entries.length,
      maxEntries: MAX_ENTRIES,
      entries: entries.slice()
    };
    if (opts.extra) b.extra = opts.extra;
    return b;
  }

  function bundleAsText(opts) {
    return JSON.stringify(bundle(opts), null, 2);
  }

  function copyBundle(opts) {
    var text = bundleAsText(opts);
    if (navigator.clipboard && navigator.clipboard.writeText) {
      return navigator.clipboard.writeText(text).then(function() {
        return { success: true, size: text.length };
      });
    }
    // Fallback for non-secure contexts
    return new Promise(function(resolve, reject) {
      var ta = document.createElement('textarea');
      ta.value = text;
      ta.style.position = 'fixed';
      ta.style.left = '-9999px';
      ta.style.top = '0';
      document.body.appendChild(ta);
      ta.focus();
      ta.select();
      try {
        var ok = document.execCommand('copy');
        document.body.removeChild(ta);
        if (ok) resolve({ success: true, size: text.length });
        else reject(new Error('copy command failed'));
      } catch (e) {
        document.body.removeChild(ta);
        reject(e);
      }
    });
  }

  function downloadBundle(opts) {
    opts = opts || {};
    var text = bundleAsText(opts);
    var blob = new Blob([text], { type: 'application/json' });
    var url = URL.createObjectURL(blob);
    var a = document.createElement('a');
    a.href = url;
    a.download = opts.filename || ('weld-diag-' + Date.now() + '.json');
    document.body.appendChild(a);
    a.click();
    setTimeout(function() {
      if (a.parentNode) a.parentNode.removeChild(a);
      URL.revokeObjectURL(url);
    }, 100);
    return { success: true, filename: a.download, size: text.length };
  }

  // -----------------------------------------------------------------
  // SUBSCRIPTIONS
  // -----------------------------------------------------------------
  function subscribe(fn) {
    if (typeof fn !== 'function') throw new Error('subscribe requires a function');
    subscribers.push(fn);
    return function unsubscribe() {
      var i = subscribers.indexOf(fn);
      if (i !== -1) subscribers.splice(i, 1);
    };
  }

  // -----------------------------------------------------------------
  // QUERY
  // -----------------------------------------------------------------
  function filter(predicate) {
    if (typeof predicate === 'string') {
      var level = predicate;
      return entries.filter(function(e) { return e.level === level; });
    }
    if (typeof predicate === 'function') return entries.filter(predicate);
    return entries.slice();
  }

  function clear() {
    var n = entries.length;
    entries.length = 0;
    return n;
  }

  // -----------------------------------------------------------------
  // INIT
  // -----------------------------------------------------------------
  installHooks();

  window.weld.diag = {
    log:   function(msg, data) { return record('info',  'manual', msg, data); },
    info:  function(msg, data) { return record('info',  'manual', msg, data); },
    warn:  function(msg, data) { return record('warn',  'manual', msg, data); },
    error: function(msg, data) { return record('error', 'manual', msg, data); },
    debug: function(msg, data) { return record('debug', 'manual', msg, data); },

    record: record,    // for custom sources: weld.diag.record('info', 'api', 'fetched', { url: '/foo' })
    entries: function() { return entries.slice(); },
    filter: filter,
    count: function() { return entries.length; },
    clear: clear,

    bundle: bundle,
    bundleAsText: bundleAsText,
    copyBundle: copyBundle,
    downloadBundle: downloadBundle,

    subscribe: subscribe,

    setMaxEntries: function(n) {
      MAX_ENTRIES = Math.max(1, n | 0);
      while (entries.length > MAX_ENTRIES) entries.shift();
      return MAX_ENTRIES;
    },
    getMaxEntries: function() { return MAX_ENTRIES; },

    version: '1.0.0'
  };
})();

</script>
