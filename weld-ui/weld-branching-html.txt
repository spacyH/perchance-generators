      <h1>weld.branching</h1>
      <span class="ver">v1.0.0</span>
    </div>
    <p class="lead">Tree-based conversation history. Branch from any message, navigate between siblings, persist the whole tree.</p>
  </div>
  <h2>Overview</h2>
<p>weld.branching tracks conversation history as a tree, not a list. Every message has a parent and zero-or-more children, so the user can fork at any point: try a different response, edit a prompt, regenerate. The active "branch" is the path from root to the current leaf.</p>
<p>This is what ChatGPT and most other chat UIs need under the hood. The plugin gives you the data structure -- you decide how to expose it in the UI (most often: a left/right arrow on each message to switch between siblings).</p>
<p>weld.branching is storage-agnostic: it works in-memory by default, persists to <code>localStorage</code> if you pass it, and can swap in IndexedDB via weld.migrate with a thin adapter.</p>

<h2>Building the tree</h2><h3>tree.append(data, parentId?)</h3>
<div class="sig" data-codeid="sig3"></div>
<p>Append a new node. If <code>parentId</code> is omitted, the new node becomes a child of the current active leaf -- normal linear progression. If <code>parentId</code> is provided, the node attaches there and the active leaf updates.</p>
<pre data-codeid="code4"></pre>
<h3>tree.branch(fromId, data)</h3>
<div class="sig" data-codeid="sig5"></div>
<p>Create a new sibling of <code>fromId</code> -- both nodes share the same parent. This is the "regenerate" / "try alternative" operation. The new node becomes the active leaf.</p>
<pre data-codeid="code6"></pre>
<h2>Reading the tree</h2><h3>tree.get(id) / .getAll() / .getRoots()</h3>
<p>Inspection primitives. Each returns a shallow copy of the node(s): <code>\{id, parentId, childIds, data, createdAt\}</code>.</p>
<h3>tree.path(toId?) / .activeBranch()</h3>
<div class="sig" data-codeid="sig7"></div>
<div class="sig" data-codeid="sig8"></div>
<p>Return the chain of nodes from a root down to <code>toId</code> (or to the current active leaf if no id given). This is what you typically render as "the conversation".</p>
<pre data-codeid="code9"></pre>
<h3>tree.siblings(id)</h3>
<p>Return all nodes that share a parent with <code>id</code> (including <code>id</code> itself). Use this to render the "1 of 3" branch picker:</p>
<pre data-codeid="code10"></pre>
<h2>Navigating</h2><h3>tree.setActive(id)</h3>
<div class="sig" data-codeid="sig11"></div>
<p>Set the active leaf directly to <code>id</code>. <code>activeBranch()</code> will now end at this node.</p>
<h3>tree.switchSibling(id, direction)</h3>
<div class="sig" data-codeid="sig12"></div>
<p>Find the next or previous sibling of <code>id</code>. <code>direction</code> can be <code>'next'</code> / <code>'prev'</code>, <code>1</code> / <code>-1</code>, or a specific index. Returns the new sibling's id, which you can pass to <code>switchTo</code>.</p>
<pre data-codeid="code13"></pre>
<h3>tree.switchTo(id)</h3>
<div class="sig" data-codeid="sig14"></div>
<p>Jump to a different branch by selecting a node. The active leaf walks down from <code>id</code> following the first-child path until it hits a leaf. Returns the new active leaf id.</p>
<p>Most UIs call this from the branch picker arrows: pick a sibling, then call <code>switchTo</code> to follow that branch all the way down to its current end.</p>
<h2>Mutation</h2><h3>tree.remove(id) / .clear()</h3>
<p><code>remove(id)</code> deletes a node and all its descendants. If the active leaf was inside that subtree, it falls back to the removed node's parent (or to the first root if it was a top-level deletion).</p>
<p><code>clear()</code> resets the entire tree.</p>
<h2>Persistence</h2><h3>serialize / deserialize</h3>
<p>For manual save/load, use <code>tree.serialize()</code> to get a plain JSON-safe object and <code>tree.deserialize(state)</code> to restore.</p>
<pre data-codeid="code15"></pre>

<h2>Behavior notes</h2><div class="note">
Removing a node cascades to all descendants. There's no undo built into the plugin -- pair with weld.undo if you need that. Snapshot the tree state via <code>tree.serialize()</code> before destructive operations.

<script>
(function(){
'use strict';

if (!window.weld) window.weld = {};
if (window.weld.branching) return; // already loaded

// -----------------------------------------------------------------
// Node id generator
// -----------------------------------------------------------------
function uid() {
  return 'n_' + Date.now().toString(36) + Math.random().toString(36).slice(2, 8);
}

// -----------------------------------------------------------------
// Tree instance factory
// -----------------------------------------------------------------
function createTree(opts) {
  opts = opts || {};
  var storage = opts.storage;
  var storageKey = opts.storageKey || 'weld-branching';

  // Internal state
  var nodes = {};      // id -> node
  var rootIds = [];    // ordered list of root node ids
  var activeId = null; // currently active leaf

  function save() {
    if (!storage) return;
    try {
      storage.setItem(storageKey, JSON.stringify({
        nodes: nodes,
        rootIds: rootIds,
        activeId: activeId
      }));
    } catch (e) { /* swallow */ }
  }

  function load() {
    if (!storage) return false;
    try {
      var raw = storage.getItem(storageKey);
      if (!raw) return false;
      var parsed = JSON.parse(raw);
      if (parsed && typeof parsed === 'object') {
        nodes = parsed.nodes || {};
        rootIds = parsed.rootIds || [];
        activeId = parsed.activeId || null;
        return true;
      }
    } catch (e) { /* swallow */ }
    return false;
  }

  // Try loading initial state
  load();

  // -----------------------------------------------------------------
  // Core: append a node
  // -----------------------------------------------------------------
  function append(data, parentId) {
    if (data === undefined) data = null;
    if (parentId === undefined) {
      // Default parent: the currently active leaf, if any
      parentId = activeId;
    }
    var id = uid();
    var node = {
      id: id,
      parentId: parentId || null,
      childIds: [],
      data: data,
      createdAt: Date.now()
    };
    nodes[id] = node;

    if (parentId && nodes[parentId]) {
      nodes[parentId].childIds.push(id);
    } else {
      // New root
      rootIds.push(id);
    }
    activeId = id;
    save();
    return id;
  }

  // -----------------------------------------------------------------
  // Branch -- explicit alias for append with parent
  // -----------------------------------------------------------------
  function branch(fromId, data) {
    if (!nodes[fromId]) throw new Error('branching: unknown node ' + fromId);
    var parentId = nodes[fromId].parentId;
    var id = append(data, parentId);
    return id;
  }

  // -----------------------------------------------------------------
  // Get / list
  // -----------------------------------------------------------------
  function get(id) {
    var n = nodes[id];
    if (!n) return null;
    return {
      id: n.id,
      parentId: n.parentId,
      childIds: n.childIds.slice(),
      data: n.data,
      createdAt: n.createdAt
    };
  }

  function getAll() {
    var out = [];
    for (var id in nodes) out.push(get(id));
    return out;
  }

  function getRoots() {
    return rootIds.map(get);
  }

  // -----------------------------------------------------------------
  // Path: from root to a given node (or active leaf)
  // -----------------------------------------------------------------
  function path(toId) {
    var target = toId || activeId;
    if (!target || !nodes[target]) return [];
    var chain = [];
    var cursor = nodes[target];
    while (cursor) {
      chain.unshift(get(cursor.id));
      cursor = cursor.parentId ? nodes[cursor.parentId] : null;
    }
    return chain;
  }

  function activeBranch() {
    return path(activeId);
  }

  function setActive(id) {
    if (!nodes[id]) throw new Error('branching: unknown node ' + id);
    activeId = id;
    save();
    return id;
  }

  function getActiveId() {
    return activeId;
  }

  // -----------------------------------------------------------------
  // Siblings -- nodes with the same parent
  // -----------------------------------------------------------------
  function siblings(id) {
    var n = nodes[id];
    if (!n) return [];
    var parentChildren;
    if (n.parentId && nodes[n.parentId]) {
      parentChildren = nodes[n.parentId].childIds;
    } else {
      parentChildren = rootIds;
    }
    return parentChildren.map(get);
  }

  // -----------------------------------------------------------------
  // Switching to a sibling -- pick by index relative to current
  // -----------------------------------------------------------------
  function switchSibling(id, direction) {
    var sibs = siblings(id);
    if (sibs.length === 0) return null;
    var idx = sibs.findIndex(function(s) { return s.id === id; });
    if (idx === -1) return null;
    var newIdx;
    if (direction === 'next' || direction === 1) {
      newIdx = (idx + 1) % sibs.length;
    } else if (direction === 'prev' || direction === -1) {
      newIdx = (idx - 1 + sibs.length) % sibs.length;
    } else if (typeof direction === 'number') {
      newIdx = Math.max(0, Math.min(sibs.length - 1, direction));
    } else {
      return null;
    }
    return sibs[newIdx].id;
  }

  // -----------------------------------------------------------------
  // Switch branch by selecting a node; updates active to that node's deepest descendant
  // along its primary (first-child) path
  // -----------------------------------------------------------------
  function switchTo(id) {
    if (!nodes[id]) throw new Error('branching: unknown node ' + id);
    // Walk down following first child until leaf
    var cursor = nodes[id];
    while (cursor.childIds.length > 0) {
      cursor = nodes[cursor.childIds[0]];
    }
    activeId = cursor.id;
    save();
    return activeId;
  }

  // -----------------------------------------------------------------
  // Remove a node and all its descendants
  // -----------------------------------------------------------------
  function remove(id) {
    var n = nodes[id];
    if (!n) return false;
    // Collect descendants
    var toRemove = [];
    var stack = [id];
    while (stack.length) {
      var cur = stack.pop();
      toRemove.push(cur);
      var node = nodes[cur];
      if (node) {
        for (var i = 0; i < node.childIds.length; i++) stack.push(node.childIds[i]);
      }
    }
    // Detach from parent
    if (n.parentId && nodes[n.parentId]) {
      nodes[n.parentId].childIds = nodes[n.parentId].childIds.filter(function(c) { return c !== id; });
    } else {
      rootIds = rootIds.filter(function(r) { return r !== id; });
    }
    // Delete all
    for (var k = 0; k < toRemove.length; k++) delete nodes[toRemove[k]];
    // Reset active if needed
    if (toRemove.indexOf(activeId) !== -1) {
      activeId = (n.parentId && nodes[n.parentId]) ? n.parentId : (rootIds[0] || null);
    }
    save();
    return true;
  }

  function clear() {
    nodes = {};
    rootIds = [];
    activeId = null;
    save();
  }

  // -----------------------------------------------------------------
  // Serialize / deserialize
  // -----------------------------------------------------------------
  function serialize() {
    return JSON.parse(JSON.stringify({
      nodes: nodes,
      rootIds: rootIds,
      activeId: activeId
    }));
  }

  function deserialize(state) {
    if (!state || typeof state !== 'object') throw new Error('branching: invalid state');
    nodes = state.nodes ? JSON.parse(JSON.stringify(state.nodes)) : {};
    rootIds = state.rootIds ? state.rootIds.slice() : [];
    activeId = state.activeId || null;
    save();
  }

  function size() {
    var n = 0;
    for (var k in nodes) n++;
    return n;
  }

  function isLeaf(id) {
    var n = nodes[id];
    return n && n.childIds.length === 0;
  }

  return {
    append: append,
    branch: branch,
    get: get,
    getAll: getAll,
    getRoots: getRoots,
    path: path,
    activeBranch: activeBranch,
    setActive: setActive,
    getActiveId: getActiveId,
    siblings: siblings,
    switchSibling: switchSibling,
    switchTo: switchTo,
    remove: remove,
    clear: clear,
    serialize: serialize,
    deserialize: deserialize,
    size: size,
    isLeaf: isLeaf
  };
}

window.weld.branching = {
  create: createTree,
  version: '1.0.0'
};

})();

</script>
