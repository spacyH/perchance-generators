  <div class="hdr-row">
      <h1>weld.summarize</h1>
      <span class="ver">v1.1.0</span>
  </div>
  <p class="lead">Keep a long conversation inside the token budget without amnesia. Where weld.prompt can only drop old turns, weld.summarize condenses them &mdash; keeping recent turns verbatim and folding the rest into a rolling, hierarchical summary.</p>
</div>
<h2>Install</h2>
<p>Import the plugin once anywhere in your generator. It stashes its API on <code>window.weld.summarize</code>. It calls <code>weld.stream</code> to write summaries and <code>weld.tokens</code> to measure the budget; both are soft dependencies, so import them too for the full experience.</p>
<pre data-codeid="install"></pre>

<h2>Quick start</h2>
<p>The common case is a <strong>rolling</strong> summarizer: feed it every message, keep the most recent turns verbatim, and let it fold older turns into a running summary as the budget fills. Build your prompt from <code>context()</code>.</p>
<pre data-codeid="quick"></pre>

<h2>summarize(input, opts)</h2>
<div class="sig" data-codeid="sig_summarize"></div>
<p>One-shot synopsis of a string or an array of messages. Returns a promise for <code>\{ ok, summary, error \}</code>. Messages may be plain strings or objects with <code>role</code>/<code>name</code> and <code>content</code>/<code>text</code>.</p>
<pre data-codeid="oneshot_ex"></pre>
<h4>opts</h4>
<table>
<tr><th>field</th><th>meaning</th></tr>
<tr><td><code>prompt</code></td><td>override the summarization instruction</td></tr>
<tr><td><code>maxWords</code></td><td>soft cap on the summary length</td></tr>
<tr><td><code>priorSummary</code></td><td>an existing summary to fold the new excerpt into</td></tr>
</table>

<h2>rolling(opts)</h2>
<div class="sig" data-codeid="sig_rolling"></div>
<p>Returns a stateful summarizer. Add messages as they happen; when the verbatim tail exceeds the token budget, the oldest turns are compressed into the running summary and dropped from the tail.</p>
<h4>opts</h4>
<table>
<tr><th>field</th><th>default</th><th>meaning</th></tr>
<tr><td><code>keepRecent</code></td><td>8</td><td>messages kept verbatim</td></tr>
<tr><td><code>tokenBudget</code></td><td>stream.budget() or 5000</td><td>compress when the tail exceeds this</td></tr>
<tr><td><code>summaryMaxWords</code></td><td>200</td><td>cap on the running summary</td></tr>
<tr><td><code>prompt</code></td><td>built-in</td><td>override the instruction</td></tr>
</table>
<h4>methods</h4>
<pre data-codeid="rolling_methods"></pre>
<div class="note">Compression is asynchronous. <code>add()</code> returns a promise that resolves once any triggered compression finishes, so you can <code>await</code> it before building the next prompt if you need the freshest context.</div>

<h2>How hierarchical compression works</h2>
<p>When the tail overflows, the older slice is summarized. If a running summary already exists, the new slice is folded <em>into</em> it &mdash; a summary of the previous summary plus the newly dropped turns. Over a long conversation this keeps one bounded synopsis instead of an ever-growing transcript, so the bot remembers the shape of the whole chat while only the recent turns stay word-for-word.</p>
<pre data-codeid="context_ex"></pre>

<h2>Persisting and restoring</h2>
<p><code>state()</code> returns a plain snapshot &mdash; the summary, the verbatim tail, and a generation counter. Save it with <code>weld.persist</code> or <code>weld.history</code>, and restore with <code>load()</code>.</p>
<pre data-codeid="persist_ex"></pre>

<h2>Helpers</h2>
<div class="sig" data-codeid="sig_estimate"></div>
<div class="sig" data-codeid="sig_join"></div>
<p><code>estimateTokens</code> uses <code>weld.tokens</code> when it is loaded, falls back to <code>weld.stream.count</code>, and finally to a chars/4 estimate. <code>joinMessages</code> renders a message array into the <code>Name: text</code> wire form.</p>
<h2>Fact extraction &amp; consolidation <span class="tag">v1.1.0</span></h2>
<div class="sig" data-codeid="sig_facts"></div>
<p><code>extractFacts(input, opts)</code> pulls discrete, durable facts out of text or a message array (rather than a prose summary) &mdash; useful for feeding <code>weld.memory</code>. <code>consolidate(summaries, opts)</code> merges many summaries into one, recursively if needed (<code>opts.maxDepth</code>, default 3; <code>opts.maxWords</code>, default 250), so a long session&rsquo;s rolling summaries collapse into a single coherent recap.</p>

<script>
(function(){
'use strict';

if (!window.weld) window.weld = {};
if (window.weld.summarize) return window.weld.summarize;

// -----------------------------------------------------------------
// weld.summarize: keep a long chat inside the context window without
// amnesia. weld.prompt can only DROP old turns when the budget is
// blown; this CONDENSES them instead.
//
//   - rolling():   a stateful summarizer. Feed it messages; it keeps
//                  the most recent N verbatim and rolls everything
//                  older into a running summary, re-summarizing
//                  (summary-of-summary) when the summary itself grows.
//   - summarize(): a one-shot. Hand it text or messages, get a synopsis.
//
// Summaries are generated through weld.stream; token counting uses
// weld.tokens when present, else a chars/4 estimate. Both are soft
// dependencies -- without weld.stream, summarize() reports an error
// rather than throwing.
// -----------------------------------------------------------------

function getStream() { return (window.weld && window.weld.stream) ? window.weld.stream : null; }
function getTokens() { return (window.weld && window.weld.tokens) ? window.weld.tokens : null; }

function estimateTokens(text) {
  var t = getTokens();
  if (t && t.isLoaded && t.isLoaded() && t.countSync) {
    var n = t.countSync(String(text || ''));
    if (n != null) return n;
  }
  var stream = getStream();
  if (stream && stream.count) {
    var sc = stream.count(String(text || ''));
    if (sc != null) return sc;
  }
  return Math.ceil(String(text || '').length / 4);
}

// Normalize a message into "Name: text" wire form for summarizing.
function lineOf(msg) {
  if (msg == null) return '';
  if (typeof msg === 'string') return msg;
  var who = msg.name || msg.role || msg.author || '';
  var body = msg.content != null ? msg.content : (msg.text != null ? msg.text : '');
  return who ? (who + ': ' + body) : String(body);
}

function joinMessages(messages) {
  if (!Array.isArray(messages)) return String(messages == null ? '' : messages);
  return messages.map(lineOf).filter(function (l) { return l && l.trim(); }).join('\n');
}

var DEFAULT_PROMPT =
  'Summarize the conversation excerpt below. Capture the durable facts, '
  + 'decisions, character details, and unresolved threads that matter for '
  + 'continuing the conversation. Write a tight third-person synopsis in past '
  + 'tense. Do not invent anything. Output only the summary.';

// Generate a synopsis of some text via weld.stream.
// Returns Promise<{ ok, summary, error }>.
function runSummary(text, opts) {
  opts = opts || {};
  var stream = getStream();
  if (!stream) {
    return Promise.resolve({ ok: false, summary: '', error: 'weld.summarize: weld.stream is not loaded' });
  }
  var body = String(text || '').trim();
  if (!body) return Promise.resolve({ ok: true, summary: '', error: null });

  var instruction = (opts.prompt || DEFAULT_PROMPT);
  if (opts.priorSummary) {
    instruction += '\n\nA summary of earlier events already exists; fold the new '
      + 'excerpt into it and return one consolidated summary.\n\nExisting summary:\n'
      + opts.priorSummary;
  }
  if (opts.maxWords) {
    instruction += '\n\nKeep the summary under about ' + opts.maxWords + ' words.';
  }
  instruction += '\n\nExcerpt:\n' + body;

  return stream.run({ instruction: instruction }).then(function (res) {
    if (res.error) return { ok: false, summary: '', error: res.error };
    return { ok: true, summary: String(res.text || '').trim(), error: null };
  });
}

// One-shot: summarize text or an array of messages.
function summarize(input, opts) {
  var text = Array.isArray(input) ? joinMessages(input) : String(input == null ? '' : input);
  return runSummary(text, opts);
}

// -----------------------------------------------------------------
// rolling(opts) -> a stateful summarizer.
//
// opts:
//   keepRecent   (number)  messages kept verbatim (default 8)
//   tokenBudget  (number)  compress when kept text exceeds this
//                          (default: weld.stream.budget() or 5000)
//   summaryMaxWords (number) cap on the running summary (default 200)
//   prompt       (string)  override the summarization instruction
//
// methods:
//   add(message)       append; may trigger async compression
//   addMany(messages)
//   compressNow()      -> Promise; force a compression pass
//   state()            -> { summary, recent, generation }
//   context()          -> "Summary of earlier events: ...\n\nName: msg..."
//   summary()          -> the running summary string
//   recent()           -> the verbatim-kept messages
//   reset()
//   onChange(fn)       subscribe to state changes; returns unsubscribe
// -----------------------------------------------------------------
function rolling(opts) {
  opts = opts || {};
  var keepRecent = opts.keepRecent || 8;
  var summaryMaxWords = opts.summaryMaxWords || 200;
  var budget = opts.tokenBudget;
  if (budget == null) {
    var stream = getStream();
    budget = (stream && stream.budget && stream.budget()) || 5000;
  }

  var recent = [];      // verbatim-kept messages
  var summary = '';     // running synopsis of everything older
  var generation = 0;   // how many compression passes have run
  var compressing = false;
  var listeners = [];

  function emit() {
    var snap = state();
    listeners.forEach(function (fn) { try { fn(snap); } catch (e) {} });
  }

  function overBudget() {
    return estimateTokens(joinMessages(recent)) > budget;
  }

  function compressNow() {
    if (compressing) return Promise.resolve(state());
    // Compress everything except the last keepRecent messages.
    if (recent.length <= keepRecent) return Promise.resolve(state());
    compressing = true;
    var cut = recent.length - keepRecent;
    var older = recent.slice(0, cut);
    var keptTail = recent.slice(cut);
    return runSummary(joinMessages(older), {
      priorSummary: summary || null,
      maxWords: summaryMaxWords,
      prompt: opts.prompt
    }).then(function (r) {
      compressing = false;
      if (r.ok && r.summary) {
        summary = r.summary;
        recent = keptTail;
        generation += 1;
        emit();
      }
      return state();
    }, function () { compressing = false; return state(); });
  }

  function maybeCompress() {
    if (recent.length > keepRecent && overBudget()) return compressNow();
    return Promise.resolve(state());
  }

  function add(message) {
    recent.push(message);
    emit();
    return maybeCompress();
  }

  function addMany(messages) {
    if (Array.isArray(messages)) recent = recent.concat(messages);
    emit();
    return maybeCompress();
  }

  function context(opts2) {
    opts2 = opts2 || {};
    var out = [];
    if (summary) out.push((opts2.label || 'Summary of earlier events') + ': ' + summary);
    var tail = joinMessages(recent);
    if (tail) out.push(tail);
    return out.join('\n\n');
  }

  function state() {
    return { summary: summary, recent: recent.slice(), generation: generation };
  }

  function reset() { recent = []; summary = ''; generation = 0; emit(); }

  function onChange(fn) {
    listeners.push(fn);
    return function () {
      var i = listeners.indexOf(fn);
      if (i !== -1) listeners.splice(i, 1);
    };
  }

  // Restore from a saved state() snapshot.
  function load(snap) {
    if (!snap) return;
    summary = snap.summary || '';
    recent = Array.isArray(snap.recent) ? snap.recent.slice() : [];
    generation = snap.generation || 0;
    emit();
  }

  return {
    add: add,
    addMany: addMany,
    compressNow: compressNow,
    context: context,
    state: state,
    load: load,
    summary: function () { return summary; },
    recent: function () { return recent.slice(); },
    reset: reset,
    onChange: onChange
  };
}

// extractFacts(input, opts) -> Promise<{ ok, facts, error }>.
// Pulls discrete, standalone facts out of text or a set of summaries as
// a JSON array of strings. Prefers weld.structured (robust JSON) and
// falls back to weld.stream with a bracket-scan parse.
var FACTS_PROMPT =
  'Extract all unique, discrete facts from the text below as a JSON array '
  + 'of strings. Each fact must be a complete, standalone statement; remove '
  + 'duplicates; preserve important names and details; keep each concise. '
  + 'Return ONLY a JSON array, e.g. ["fact one.", "fact two."]. If there is '
  + 'nothing worth keeping, return [].';

function extractFacts(input, opts) {
  opts = opts || {};
  var text = Array.isArray(input) ? joinMessages(input) : String(input == null ? '' : input);
  if (!text.trim()) return Promise.resolve({ ok: true, facts: [], error: null });
  var instruction = (opts.prompt || FACTS_PROMPT) + '\n\nText:\n' + text;

  var structured = (window.weld && window.weld.structured) ? window.weld.structured : null;
  if (structured) {
    return structured.generate({ instruction: instruction, startWith: '[', shape: 'a JSON array of short factual strings' })
      .then(function (r) {
        if (r.ok && Array.isArray(r.value)) return { ok: true, facts: r.value.map(String), error: null };
        return { ok: false, facts: [], error: r.error || 'no facts extracted' };
      });
  }
  var stream = getStream();
  if (!stream) return Promise.resolve({ ok: false, facts: [], error: 'weld.summarize: weld.stream is not loaded' });
  return stream.run({ instruction: instruction, startWith: '[' }).then(function (res) {
    if (res.error) return { ok: false, facts: [], error: res.error };
    var list = [];
    try {
      var raw = res.text, open = raw.indexOf('['), close = raw.lastIndexOf(']');
      if (open !== -1 && close > open) list = JSON.parse(raw.slice(open, close + 1));
    } catch (e) { list = []; }
    if (!Array.isArray(list)) list = [];
    return { ok: true, facts: list.map(String), error: null };
  });
}

// consolidate(summaries, opts) -> Promise<{ ok, summary, depth, error }>.
// Merges several summaries (strings or { summary, depth }) into one, with
// a hard depth cap so repeated re-summarization can't recurse forever.
// opts: { maxDepth (default 3), maxWords (default 250), prompt }
function consolidate(summaries, opts) {
  opts = opts || {};
  var maxDepth = opts.maxDepth != null ? opts.maxDepth : 3;
  var list = (summaries || []).map(function (s) {
    return (typeof s === 'string') ? { summary: s, depth: 0 } : { summary: String(s.summary || ''), depth: s.depth || 0 };
  }).filter(function (s) { return s.summary.trim(); });

  if (!list.length) return Promise.resolve({ ok: true, summary: '', depth: 0, error: null });
  var inDepth = list.reduce(function (m, s) { return Math.max(m, s.depth); }, 0);
  var outDepth = inDepth + 1;
  if (inDepth >= maxDepth) {
    // refuse to recurse further: concatenate as-is
    return Promise.resolve({ ok: true, summary: list.map(function (s) { return s.summary; }).join('\n\n'), depth: inDepth, error: null, capped: true });
  }
  var combined = list.map(function (s) { return s.summary; }).join('\n\n---\n\n');
  return runSummary(combined, {
    prompt: opts.prompt || 'Merge the following summaries into one coherent, non-redundant summary that preserves every important fact, name, and unresolved thread.',
    maxWords: opts.maxWords || 250
  }).then(function (r) {
    if (!r.ok) return { ok: false, summary: '', depth: outDepth, error: r.error };
    return { ok: true, summary: r.summary, depth: outDepth, error: null };
  });
}

window.weld.summarize = {
  summarize: summarize,
  rolling: rolling,
  extractFacts: extractFacts,
  consolidate: consolidate,
  estimateTokens: estimateTokens,
  joinMessages: joinMessages,
  version: '1.1.0'
};

return window.weld.summarize;
})();
