  <h2>Overview</h2>
<p>weld.chatui provides the rendering scaffolding for chat-style interfaces. It handles the parts that are tedious to reimplement but not particularly differentiating: message bubble layout, role-based styling, streaming responses with a live cursor, typing indicators, and intelligent auto-scroll that respects when the user has scrolled away.</p>
<p>What it deliberately does not handle: where the messages come from, what they mean, how to persist them. Those are your generator's concerns. weld.chatui is the view layer; you bring the model.</p>
<p>Soft dependencies: weld.markdown (used automatically if loaded; renders message content as markdown) and weld.ui (provides CSS variables for theming).</p>

<h2>Configuration</h2>
<p>All options to <code>create()</code> are optional except <code>container</code>.</p>
<ul>
<li><code>container</code>: CSS selector or element. Required.</li>
<li><code>onSend</code>: <code>(text, chatApi) =&gt; void</code>. Called when the user hits Send or Enter in the input area.</li>
<li><code>input</code>: pass <code>false</code> to skip the built-in input area (e.g. you want a custom one). Default true.</li>
<li><code>sendLabel</code>: text on the send button. Default <code>'Send'</code>.</li>
<li><code>placeholder</code>: input placeholder text. Default <code>'Type a message...'</code>.</li>
<li><code>markdown</code>: <code>'auto'</code> (use weld.markdown if loaded), <code>true</code> (force), <code>false</code> (plain text), or a function <code>(text) =&gt; html</code>. Default <code>'auto'</code>.</li>
<li><code>autoScroll</code>: scroll to bottom on new messages. The auto-scroll pauses if the user has scrolled up manually. Default true.</li>
<li><code>showAvatars</code>: render avatar circles. Default true.</li>
<li><code>showLabels</code>: render role labels under bubbles. Default true.</li>
<li><code>roles</code>: override per-role settings. See below.</li>
</ul>
<h2>Role configuration</h2>
<p>The three default roles are <code>user</code>, <code>assistant</code>, and <code>system</code>. Pass a <code>roles</code> map to override or add roles:</p>
<pre data-codeid="code4"></pre>

<p>The <code>side</code> field is one of <code>'left'</code>, <code>'right'</code>, or <code>'center'</code>. Center-aligned messages are styled as italic dashed-border notes -- useful for system or narrator content.</p>
<h2>Sending and receiving</h2><h3>chat.addMessage(msg)</h3>
<div class="sig" data-codeid="sig5"></div>
<p>Append a complete message to the conversation. Returns the message id (auto-generated if not provided). Auto-scrolls to bottom unless the user has scrolled up.</p>
<pre data-codeid="code6"></pre>
<h3>chat.streamMessage(opts?)</h3>
<div class="sig" data-codeid="sig7"></div>
<p>Start a message that will receive content incrementally. The bubble shows a blinking cursor while streaming. The returned handle has four methods:</p>
<ul>
<li><code>append(chunk)</code>: add a chunk of text. Re-renders the bubble.</li>
<li><code>replace(text)</code>: replace the full content (e.g. server sends complete state each time).</li>
<li><code>finish(finalText?)</code>: mark streaming done. Removes the cursor. Optionally pass a final value to set.</li>
<li><code>cancel()</code>: stop streaming without finalizing. Bubble stays with whatever content was last appended.</li>
</ul>
<pre data-codeid="code8"></pre>
<h3>chat.updateMessage(id, content) / .removeMessage(id) / .clearMessages()</h3>
<p>Standard manipulation. <code>updateMessage</code> replaces an existing message's content (useful for edit functionality). <code>removeMessage</code> drops it from the DOM and the messages array. <code>clearMessages</code> resets to an empty chat.</p>
<h2>Typing indicator</h2>
<div class="sig" data-codeid="sig9"></div>
<p>Animated three-dot indicator. Use while waiting for a response that hasn't started streaming yet:</p>
<pre data-codeid="code10"></pre>
<p>The indicator appears at the bottom of the message list and gets pushed aside when a new message arrives. It's a separate DOM element from the messages -- showing/hiding it doesn't touch the message list.</p>
<h2>Scrolling behavior</h2>
<p>Auto-scroll defaults to on but respects user intent. If the user scrolls up to read older messages, new messages won't yank the view back down -- they accumulate at the bottom and the user can scroll back when they choose. The "scrolled up" state is automatically detected from the messages container's scroll position.</p>
<p>To programmatically jump to the bottom (e.g. after the user clicks a "scroll to bottom" button), call <code>chat.scrollToBottom()</code>. This resets the user-scrolled-up flag.</p>
<h2>Markdown rendering</h2>
<p>When <code>markdown</code> is <code>'auto'</code> (the default) or <code>true</code>, weld.chatui calls <code>weld.markdown.render()</code> on each message's content. If weld.markdown isn't loaded, content is rendered as plain text with newlines preserved as <code>&lt;br&gt;</code>.</p>
<p>For streaming messages, the bubble is re-rendered on every <code>append</code> call. This means markdown like fenced code blocks renders correctly once the closing fence arrives.</p>
<p>To use a custom renderer:</p>
<pre data-codeid="code11"></pre>
<p>To switch rendering modes at runtime, call <code>chat.setMarkdown(opt)</code>. All existing messages get re-rendered.</p>

<h3>Headless usage</h3>
<p>If you want full control over the input UI but still want the rendering, set <code>input: false</code> and call <code>addMessage</code> / <code>streamMessage</code> directly from your own event handlers.</p>

function createChat(opts) {
  opts = opts || {};
  injectCss();

  var container = resolveContainer(opts.container);
  if (!container) throw new Error('weld.chatui.create: container not found');

  var roles = Object.assign({}, DEFAULT_ROLES, opts.roles || {});
  var autoScroll = opts.autoScroll !== false;
  var showAvatars = opts.showAvatars !== false;
  var showLabels = opts.showLabels !== false;
  var markdownOpt = opts.markdown == null ? 'auto' : opts.markdown;

  function getMarkdownRenderer() {
    if (markdownOpt === false) return null;
    if (typeof markdownOpt === 'function') return markdownOpt;
    if (markdownOpt === 'auto' || markdownOpt === true) {
      if (window.weld && window.weld.markdown) {
        return function(text) { return window.weld.markdown.render(text); };
      }
    }
    return null;
  }

  // Build skeleton
  container.classList.add('weld-chat');
  container.innerHTML = '';
  var messagesEl = document.createElement('div');
  messagesEl.className = 'weld-chat-messages';
  container.appendChild(messagesEl);

  var inputEl = null;
  if (opts.input !== false) {
    inputEl = document.createElement('div');
    inputEl.className = 'weld-chat-input';
    var ta = document.createElement('textarea');
    ta.placeholder = opts.placeholder || 'Type a message...';
    ta.rows = 1;
    ta.addEventListener('input', function() {
      ta.style.height = 'auto';
      ta.style.height = Math.min(ta.scrollHeight, 140) + 'px';
    });

    btn.textContent = opts.sendLabel || 'Send';
    function doSend() {
      var text = ta.value.trim();
      if (!text) return;
      ta.value = '';
      ta.style.height = 'auto';
      if (typeof opts.onSend === 'function') {
        try { opts.onSend(text, api); } catch (e) { /* swallow */ }
      }
    }
    btn.addEventListener('click', doSend);
    ta.addEventListener('keydown', function(e) {
      if (e.key === 'Enter' && !e.shiftKey) {
        e.preventDefault();
        doSend();
      }
    });
    inputEl.appendChild(ta);
    inputEl.appendChild(btn);
    container.appendChild(inputEl);
  }

  // State
  var messages = [];
  var messageEls = {};
  var typingEl = null;

  // Track if user has scrolled away from bottom
  var userScrolledUp = false;
  messagesEl.addEventListener('scroll', function() {
    var atBottom = messagesEl.scrollTop + messagesEl.clientHeight >= messagesEl.scrollHeight - 10;
    userScrolledUp = !atBottom;
  });

  function maybeScrollToBottom() {
    if (autoScroll && !userScrolledUp) {
      messagesEl.scrollTop = messagesEl.scrollHeight;
    }
  }

  function scrollToBottom() {
    userScrolledUp = false;
    messagesEl.scrollTop = messagesEl.scrollHeight;
  }

  function renderContent(text) {
    var renderer = getMarkdownRenderer();
    if (renderer) {
      try { return renderer(text); } catch (e) { /* fall through */ }
    }
    // Plain text: escape and convert newlines to <br>
    return escapeHtml(text).replace(/\n/g, '<br>');
  }

  function escapeHtml(s) {
    return String(s).replace(/&/g, '&amp;').replace(/</g, '&lt;').replace(/>/g, '&gt;');
  }

  function buildRowEl(msg) {
    var role = roles[msg.role] || roles.user;
    var row = document.createElement('div');
    row.className = 'weld-chat-row side-' + role.side;
    row.dataset.messageId = msg.id;
    row.dataset.role = msg.role;

    if (showAvatars && role.side !== 'center') {
      var av = document.createElement('div');
      av.className = 'weld-chat-avatar';
      av.style.background = role.color;
      av.textContent = role.initial;
      row.appendChild(av);
    }

    var col = document.createElement('div');
    col.style.flex = '1';
    col.style.minWidth = '0';

    var bubble = document.createElement('div');
    bubble.className = 'weld-chat-bubble';
    bubble.innerHTML = renderContent(msg.content);
    col.appendChild(bubble);

    if (showLabels && role.side !== 'center') {
      var meta = document.createElement('div');
      meta.className = 'weld-chat-meta';
      meta.innerHTML = '<span class="role-label">' + escapeHtml(role.label) + '</span>';
      if (msg.timestamp) {
        var ts = new Date(msg.timestamp).toLocaleTimeString([], { hour: '2-digit', minute: '2-digit' });
        meta.innerHTML += '<span>' + ts + '</span>';
      }
      col.appendChild(meta);
    }

    row.appendChild(col);
    return row;
  }

  function addMessage(msg) {
    msg = msg || {};
    if (!msg.role) msg.role = 'user';
    if (msg.content == null) msg.content = '';
    if (!msg.id) msg.id = uid();
    if (!msg.timestamp) msg.timestamp = Date.now();

    messages.push(msg);
    var rowEl = buildRowEl(msg);
    messageEls[msg.id] = rowEl;
    // Insert before typing indicator if present
    if (typingEl && typingEl.parentNode) {
      messagesEl.insertBefore(rowEl, typingEl);
    } else {
      messagesEl.appendChild(rowEl);
    }
    maybeScrollToBottom();
    return msg.id;
  }

  function updateMessage(id, content) {
    var msg = messages.find(function(m) { return m.id === id; });
    if (!msg) return false;
    msg.content = content;
    var rowEl = messageEls[id];
    if (rowEl) {
      var bubble = rowEl.querySelector('.weld-chat-bubble');
      if (bubble) bubble.innerHTML = renderContent(content);
    }
    maybeScrollToBottom();
    return true;
  }

  function removeMessage(id) {
    var idx = messages.findIndex(function(m) { return m.id === id; });
    if (idx === -1) return false;
    messages.splice(idx, 1);
    var rowEl = messageEls[id];
    if (rowEl && rowEl.parentNode) rowEl.parentNode.removeChild(rowEl);
    delete messageEls[id];
    return true;
  }

  function clearMessages() {
    messages = [];
    messageEls = {};
    messagesEl.innerHTML = '';
    if (typingEl) messagesEl.appendChild(typingEl);
  }

  function streamMessage(opts) {
    opts = opts || {};
    var id = addMessage({ role: opts.role || 'assistant', content: '', id: opts.id });
    var rowEl = messageEls[id];
    rowEl.classList.add('weld-chat-streaming');
    var content = '';
    var msg = messages.find(function(m) { return m.id === id; });

    return {
      id: id,
      append: function(chunk) {
        content += chunk;
        msg.content = content;
        var bubble = rowEl.querySelector('.weld-chat-bubble');
        if (bubble) bubble.innerHTML = renderContent(content);
        maybeScrollToBottom();
      },
      replace: function(newContent) {
        content = String(newContent);
        msg.content = content;
        var bubble = rowEl.querySelector('.weld-chat-bubble');
        if (bubble) bubble.innerHTML = renderContent(content);
        maybeScrollToBottom();
      },
      finish: function(finalContent) {
        if (finalContent != null) {
          content = String(finalContent);
          msg.content = content;
          var bubble = rowEl.querySelector('.weld-chat-bubble');
          if (bubble) bubble.innerHTML = renderContent(content);
        }
        rowEl.classList.remove('weld-chat-streaming');
        return msg;
      },
      cancel: function() {
        rowEl.classList.remove('weld-chat-streaming');
      }
    };
  }

  function showTyping() {
    if (typingEl) return;
    typingEl = document.createElement('div');
    typingEl.className = 'weld-chat-typing';
    typingEl.innerHTML =
      '<div class="weld-chat-typing-dot"></div>' +
      '<div class="weld-chat-typing-dot"></div>' +
      '<div class="weld-chat-typing-dot"></div>';
    messagesEl.appendChild(typingEl);
    maybeScrollToBottom();
  }

  function hideTyping() {
    if (typingEl && typingEl.parentNode) {
      typingEl.parentNode.removeChild(typingEl);
    }
    typingEl = null;
  }

  function getMessages() {
    return messages.slice();
  }

  function setMarkdown(opt) {
    markdownOpt = opt;
    // Re-render all messages
    for (var i = 0; i < messages.length; i++) {
      var m = messages[i];
      var rowEl = messageEls[m.id];
      if (rowEl) {
        var bubble = rowEl.querySelector('.weld-chat-bubble');
        if (bubble) bubble.innerHTML = renderContent(m.content);
      }
    }
  }

  function focus() {
    if (inputEl) {
      var ta = inputEl.querySelector('textarea');
      if (ta) ta.focus();
    }
  }

  var api = {
    addMessage: addMessage,
    updateMessage: updateMessage,
    removeMessage: removeMessage,
    clearMessages: clearMessages,
    streamMessage: streamMessage,
    showTyping: showTyping,
    hideTyping: hideTyping,
    getMessages: getMessages,
    scrollToBottom: scrollToBottom,
    setMarkdown: setMarkdown,
    focus: focus,
    container: container,
    messagesEl: messagesEl,
    inputEl: inputEl
  };
  return api;
}

window.weld.chatui = {
  create: createChat,
  version: '1.0.0'
};

})();
