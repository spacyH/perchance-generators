      <h1>weld.spamguard</h1>
      <span class="ver">v1.0.0</span>
    </div>
    <p class="lead">Multi-channel spam detection with text fingerprinting and Jaccard similarity. The plugin detects; your callbacks decide what to do.</p>
  </div>
  <h2>Overview</h2>
<p>weld.spamguard detects the canonical multi-channel spam pattern -- the same or near-identical message posted to several different channels by the same user inside a short time window -- and routes the result through callbacks you control. The plugin itself never bans or deletes anything; it tells you who looks like a spammer and you decide what to do.</p>
<p>Detection uses two layers: a normalised text fingerprint (lowercase, NFKD, diacritics stripped, emoji removed, repeated characters dampened, whitespace collapsed) and Jaccard similarity on the resulting word sets. Spammers who tweak a word or two between channels still get caught. Whitelisted user ids (moderators, announcement accounts) are skipped entirely.</p>
<p>Pairs naturally with weld.chatui: feed every incoming message into <code>guard.report</code> and use <code>chat.removeMessage</code> as the <code>deleteFn</code>.</p>

<h2>Configuration</h2>
<p>All options to <code>create()</code> are optional with sensible defaults.</p>
<ul>
<li><code>channelThreshold</code> (3): how many different channels must see the same message before it's spam.</li>
<li><code>windowMs</code> (60000): sliding time window for the channel count.</li>
<li><code>similarityThreshold</code> (0.92): Jaccard similarity to call two messages "the same". 1.0 means exact match, 0.85 means most words shared, 0.5 means about half.</li>
<li><code>minMessageLength</code> (5): messages shorter than this are ignored. Prevents "lol" / "hi" from triggering.</li>
<li><code>maxHistoryPerUser</code> (50): cap on per-user message history. Older entries get pruned.</li>
<li><code>autoBan</code> (true): record the ban internally and call <code>banFn</code> if provided. Set false to surface spam reports for human review.</li>
<li><code>autoDelete</code> (true): call <code>deleteFn</code> for each flagged message.</li>
<li><code>banDurationMs</code> (0): if &gt; 0, the ban expires after this many ms. 0 means permanent (until you call <code>unban</code>).</li>
<li><code>whitelistUserIds</code> (\[\]): user ids that are never flagged. Use for moderators or accounts that legitimately cross-post.</li>
<li><code>banFn</code>, <code>deleteFn</code>: your callbacks. Both can be async; the plugin awaits them.</li>
<li><code>onSpamDetected(spam)</code>: pre-action hook. Return <code>false</code> to veto the ban (use for additional checks). Otherwise the standard flow runs.</li>
<li><code>onBan(userId, info)</code>: notification hook after a ban happens. For logging or UI updates.</li>
</ul>

<h2>Algorithm details</h2><h3>Fingerprinting</h3>
<p>Every message text goes through the same normalisation pipeline before comparison:</p>
<ol>
<li>Lowercase</li>
<li>NFKD normalize, strip combining diacritics</li>
<li>Remove emoji (Unicode presentation + extended pictographic)</li>
<li>Replace non-letter non-number characters with spaces</li>
<li>Dampen 3+ consecutive identical characters to 2 (e.g. "soooo" -> "soo")</li>
<li>Collapse whitespace, trim</li>
</ol>
<p>The result is what's compared. Two messages with identical fingerprints are considered exact matches; messages with high Jaccard similarity on the word set are considered near-identical.</p>
<h3>Jaccard similarity</h3>
<p>For two word sets A and B, similarity is <code>|A intersect B| / |A union B|</code>. Range is 0 (no shared words) to 1 (identical sets). At the default threshold of 0.92, "join my discord at xxx" and "join my discord today at xxx" would match (only one word differs from a 5-word set).</p>
<p>The <code>fingerprint</code> and <code>jaccard</code> functions are exposed on the top-level <code>weld.spamguard</code> object if you want to use them directly without going through a guard.</p>

<script>
(function(){
'use strict';

if (!window.weld) window.weld = {};
if (window.weld.spamguard) return; // already loaded

// -----------------------------------------------------------------
// Multi-channel chat spam guard.
// Detects copy-paste spam across channels using:
//   1. Normalised text fingerprint (lowercase, NFKD, emoji + punct stripped,
//      char-repeat dampened, whitespace collapsed)
//   2. Token-set Jaccard similarity for near-identical messages
// Cross-channel threshold + sliding time window controls ban trigger.
// -----------------------------------------------------------------

function fingerprint(text) {
  if (typeof text !== 'string') return '';
  return text
    .toLowerCase()
    .normalize('NFKD')
    .replace(/[\u0300-\u036f]/g, '')
    .replace(/[\p{Emoji_Presentation}\p{Extended_Pictographic}]/gu, '')
    .replace(/[^\p{L}\p{N}\s]/gu, ' ')
    .replace(/(.)\1{2,}/g, '$1$1')
    .replace(/\s+/g, ' ')
    .trim();
}

// Jaccard similarity on word-token sets, 0 (disjoint) to 1 (identical)
function jaccard(a, b) {
  if (a === b) return 1;
  if (!a || !b) return 0;
  var A = new Set(a.split(' '));
  var B = new Set(b.split(' '));
  var inter = 0;
  var itr = A.values();
  var step;
  while (!(step = itr.next()).done) {
    if (B.has(step.value)) inter++;
  }
  var union = A.size + B.size - inter;
  return union === 0 ? 0 : inter / union;
}

function similarEnough(a, b, threshold) {
  if (a === b) return true;
  return jaccard(a, b) >= threshold;
}

function createGuard(opts) {
  opts = opts || {};
  var DEFAULTS = {
    channelThreshold: 3,
    windowMs: 60000,
    similarityThreshold: 0.92,
    minMessageLength: 5,
    maxHistoryPerUser: 50,
    autoBan: true,
    autoDelete: true,
    banDurationMs: 0,
    whitelistUserIds: [],
    banFn: null,
    deleteFn: null,
    onSpamDetected: null,
    onBan: null
  };
  var merged = Object.assign({}, DEFAULTS, opts);

  var history = new Map();   // userId -> array of message records
  var banned  = new Map();   // userId -> ban record (reason / bannedAt / expiresAt)
  var stats   = { tracked: 0, banned: 0, totalReports: 0, totalBans: 0, totalDeletes: 0 };

  function pruneOld(userId, now) {
    var arr = history.get(userId);
    if (!arr) return;
    var cutoff = now - merged.windowMs;
    var i = 0;
    while (i < arr.length && arr[i].timestamp < cutoff) i++;
    if (i > 0) arr.splice(0, i);
    if (arr.length > merged.maxHistoryPerUser) {
      arr.splice(0, arr.length - merged.maxHistoryPerUser);
    }
  }

  function checkBanExpiry(userId, now) {
    var b = banned.get(userId);
    if (b && b.expiresAt && b.expiresAt <= now) {
      banned.delete(userId);
      return false;
    }
    return !!b;
  }

  function classify(userId, fp, now) {
    var arr = history.get(userId) || [];
    var matchedByChannel = new Map();
    for (var i = 0; i < arr.length; i++) {
      var m = arr[i];
      if (similarEnough(m.fp, fp, merged.similarityThreshold)) {
        if (!matchedByChannel.has(m.channel)) matchedByChannel.set(m.channel, m);
      }
    }
    if (matchedByChannel.size >= merged.channelThreshold) {
      var matches = Array.from(matchedByChannel.values());
      return {
        fingerprint: fp,
        channels: Array.from(matchedByChannel.keys()),
        messages: matches,
        firstSeen: Math.min.apply(null, matches.map(function(m) { return m.timestamp; })),
        lastSeen: now
      };
    }
    return null;
  }

  // -------- PUBLIC: report a new message --------
  function report(input) {
    return Promise.resolve().then(function() {
      stats.totalReports++;
      if (!input || !input.userId || !input.channel) {
        return { ok: false, reason: 'missing userId or channel' };
      }
      var channel = input.channel;
      var userId = input.userId;
      var messageId = input.messageId;
      var text = input.text;

      if (merged.whitelistUserIds.indexOf(userId) !== -1) {
        return { ok: true, whitelisted: true };
      }

      var now = input.timestamp || Date.now();
      if (checkBanExpiry(userId, now)) {
        return { ok: false, banned: true };
      }

      if (typeof text !== 'string' || text.length < merged.minMessageLength) {
        return { ok: true };
      }
      var fp = fingerprint(text);
      if (!fp) return { ok: true };

      pruneOld(userId, now);
      if (!history.has(userId)) history.set(userId, []);
      var arr = history.get(userId);
      arr.push({ channel: channel, fp: fp, text: text, messageId: messageId, timestamp: now });
      stats.tracked = history.size;

      var spam = classify(userId, fp, now);
      if (!spam) return { ok: true };

      // Include the triggering message itself
      var alreadyHas = false;
      for (var i = 0; i < spam.messages.length; i++) {
        if (spam.messages[i].channel === channel) { alreadyHas = true; break; }
      }
      if (!alreadyHas) {
        spam.channels.push(channel);
        spam.messages.push({ channel: channel, fp: fp, text: text, messageId: messageId, timestamp: now });
      }

      var verdict = Promise.resolve(true);
      if (typeof merged.onSpamDetected === 'function') {
        verdict = Promise.resolve(merged.onSpamDetected(spam));
      }

      return verdict.then(function(v) {
        if (v === false) return { ok: true, vetoed: true, spam: spam };

        var actions = Promise.resolve();
        if (merged.autoBan) {
          var expiresAt = merged.banDurationMs > 0 ? now + merged.banDurationMs : null;
          banned.set(userId, { reason: 'cross-channel-spam', bannedAt: now, expiresAt: expiresAt });
          stats.banned++;
          stats.totalBans++;
          if (typeof merged.banFn === 'function') {
            actions = actions.then(function() {
              return Promise.resolve(merged.banFn(userId, { reason: 'cross-channel-spam', spam: spam }))
                .catch(function(e) { console.error('[weld.spamguard] banFn threw:', e); });
            });
          }
          if (typeof merged.onBan === 'function') {
            actions = actions.then(function() {
              return Promise.resolve(merged.onBan(userId, { reason: 'cross-channel-spam', spam: spam }))
                .catch(function() {});
            });
          }
        }

        if (merged.autoDelete && typeof merged.deleteFn === 'function') {
          spam.messages.forEach(function(m) {
            actions = actions.then(function() {
              return Promise.resolve(merged.deleteFn(m.messageId, m.channel))
                .then(function() { stats.totalDeletes++; })
                .catch(function(e) { console.error('[weld.spamguard] deleteFn threw:', e, m); });
            });
          });
        }

        return actions.then(function() {
          history.delete(userId);
          return { ok: false, banned: true, spam: spam };
        });
      });
    });
  }

  function ban(userId, reason) {
    banned.set(userId, { reason: reason || 'manual', bannedAt: Date.now(), expiresAt: null });
    stats.banned++;
    stats.totalBans++;
  }

  function unban(userId) {
    if (banned.delete(userId)) stats.banned = Math.max(0, stats.banned - 1);
  }

  function isBanned(userId) {
    return checkBanExpiry(userId, Date.now());
  }

  function bannedUsers() {
    var now = Date.now();
    return Array.from(banned.keys()).filter(function(id) { return checkBanExpiry(id, now); });
  }

  function getHistory(userId) {
    return (history.get(userId) || []).slice();
  }

  function flush() {
    history.clear();
    stats.tracked = 0;
  }

  function getStats() {
    return Object.assign({}, stats, {
      tracked: history.size,
      banned: bannedUsers().length
    });
  }

  return {
    report: report,
    ban: ban,
    unban: unban,
    isBanned: isBanned,
    bannedUsers: bannedUsers,
    history: getHistory,
    flush: flush,
    options: merged,
    stats: getStats,
    fingerprint: fingerprint,
    jaccard: jaccard
  };
}

window.weld.spamguard = {
  create: createGuard,
  fingerprint: fingerprint,
  jaccard: jaccard,
  version: '1.0.0'
};

})();

</script>
