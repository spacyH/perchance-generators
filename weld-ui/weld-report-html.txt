      <h1>weld.report</h1>
      <span class="ver">v1.0.0</span>
  </div>
  <p class="lead">A drop-in superset of Perchance's bug-report-plugin: the same mandatory-encryption, expiring-URL workflow and identical API, plus a de-duplicated timestamped error log, a human-readable summary, and a weld.monitor platform fingerprint. Composes with weld.diag and weld.monitor.</p>

<h2>A drop-in superset of bug-report-plugin</h2>
<p>Perchance's official <code>bug-report-plugin</code> captures the reporter's browser and recent errors, encrypts the payload with the developer's public key, and uploads it as a URL that self-deletes after six months. weld.report keeps that exact workflow &mdash; and the exact API &mdash; so it swaps in unchanged, then adds the things that make a report actually fixable.</p>
<pre data-codeid="install"></pre>
<div class="note">Encryption is <strong>mandatory</strong>, by design: a raw report would leak the reporter's browser, device, and storage details. Generate a key pair at <code>perchance.org/public-key-encryption-tool</code> and keep the private key; only you can read submitted reports.</div>

<h2>createTemporaryDebugInfoUrl(opts) <span class="tag">official, unchanged</span></h2>
<div class="sig" data-codeid="sig_url"></div>
<p>Identical signature to the official plugin: pass <code>\{ publicKey, customData \}</code>, get back a Promise of an expiring URL (or <code>null</code> on failure). Encrypts the report with <code>secret-plugin</code> and uploads it via <code>upload-plugin</code> with a six-month expiry. Anything written against the canonical plugin works as-is.</p>
<pre data-codeid="url_ex"></pre>

<h2>initAutoErrorCapture() <span class="tag">official, smarter</span></h2>
<div class="sig" data-codeid="sig_cap"></div>
<p>Starts capturing uncaught errors, unhandled rejections, and <code>console.error</code>. If <code>weld.diag</code> is loaded, weld.report <em>subscribes to its existing capture stream</em> instead of double-hooking the console; otherwise it installs its own minimal hooks. Either way the data flows into a de-duplicated log.</p>

<h2>What it adds</h2>
<p><strong>A timestamped, de-duplicated error log.</strong> The official plugin keeps a flat array of the last six console errors; one noisy error firing two hundred times buries everything else. weld.report collapses each distinct <code>source + message</code> into one row with an occurrence count and first/last-seen timestamps.</p>
<div class="sig" data-codeid="sig_log"></div>
<pre data-codeid="log_ex"></pre>

<p><strong>A human-readable summary.</strong> <code>summary()</code> returns a compact digest &mdash; generator, browser, viewport, platform fingerprint, and the top errors with counts &mdash; that you can show the user or log to the console. The official plugin only ever emits an opaque encrypted blob.</p>
<div class="sig" data-codeid="sig_sum"></div>

<p><strong>A weld.monitor platform fingerprint.</strong> Hand weld.report a <code>weld.monitor</code> run and its fingerprint rides along in every report, so you can tell "this happened during a Cloudflare-proxy outage" from a genuine app bug.</p>
<div class="sig" data-codeid="sig_fp"></div>
<pre data-codeid="fp_ex"></pre>
<div class="note">Composes with the suite: reuses <code>weld.diag</code> for capture and environment, <code>weld.monitor</code> for the fingerprint, and the platform's <code>secret-plugin</code> / <code>upload-plugin</code> for the encrypted expiring upload. Drop it in where you'd use the official plugin and you lose nothing, gain triage.</div>

<script>
(function(){
'use strict';

if (!window.weld) window.weld = {};
if (window.weld.report) return window.weld.report;

// -----------------------------------------------------------------
// weld.report: bug reports that are actually fixable.
//
// Perchance's official bug-report-plugin captures the browser/device and
// a flat list of the last six console errors, encrypts it with the dev's
// public key, and uploads it as a URL that self-deletes after six months.
// That privacy stance is exactly right (a raw report would leak the
// reporter's browser, device, storage, etc.), so weld.report keeps it --
// encryption is mandatory and the API is identical:
//
//   await weld.report.createTemporaryDebugInfoUrl({ publicKey, customData });
//   weld.report.initAutoErrorCapture();
//
// What it adds, by composing the rest of the suite:
//   * a TIMESTAMPED, DE-DUPLICATED error log (not a flat array of 6) with
//     occurrence counts -- so a single error firing 200x reads as one row;
//   * a HUMAN-READABLE summary (weld.report.summary()) the app can show the
//     user or log, instead of only an opaque encrypted blob;
//   * a weld.monitor FINGERPRINT embedded in the report, so the dev can
//     tell "this happened during a Cloudflare-proxy outage" from a real bug.
//
// It reuses weld.diag's capture + environment bundle and weld.monitor's
// fingerprint when they're loaded, and self-captures otherwise. Encryption
// and expiring upload mirror the official plugin's platform calls exactly.
// -----------------------------------------------------------------

var META_KEY = '__weldReportMeta_b9f3';
if (!Reflect.get(window, META_KEY)) Reflect.set(window, META_KEY, {});
var meta = Reflect.get(window, META_KEY);

var SB = String.fromCharCode(91), SC = String.fromCharCode(93); // open/close square, built at runtime

var deps = { diag: null, monitor: null };
function attach(d) {
  if (!d) return;
  if (d.diag) deps.diag = d.diag;
  if (d.monitor) deps.monitor = d.monitor;
}
function diag() { return deps.diag || (window.weld && window.weld.diag) || null; }
function monitor() { return deps.monitor || (window.weld && window.weld.monitor) || null; }

function nowIso() { try { return new Date().toISOString(); } catch (e) { return '' + Date.now(); } }
function safeStr(v) {
  if (v == null) return '';
  if (typeof v === 'string') return v;
  try { return JSON.stringify(v); } catch (e) { return String(v); }
}

// ---- error log: timestamped + de-duplicated -------------------------
// Each distinct (source + message) is one row carrying first/last seen and
// an occurrence count, so a noisy repeated error doesn't drown the report.
function ensureLog() {
  if (!meta.log) meta.log = { rows: {}, order: new Array(), max: 50 };
  return meta.log;
}
function logError(source, message, extra) {
  var log = ensureLog();
  var msg = String(message == null ? '' : message);
  var key = source + '\u0001' + msg;
  var row = Reflect.get(log.rows, key);
  var t = nowIso();
  if (row) {
    row.count++;
    row.lastSeen = t;
    if (extra && !row.extra) row.extra = extra;
  } else {
    row = { source: source, message: msg, count: 1, firstSeen: t, lastSeen: t, extra: extra || null };
    Reflect.set(log.rows, key, row);
    log.order.push(key);
    while (log.order.length > log.max) { var drop = log.order.shift(); Reflect.deleteProperty(log.rows, drop); }
  }
  return row;
}
// Snapshot the de-duplicated log, newest activity first.
function errorLog() {
  var log = ensureLog();
  return log.order.map(function (k) { return Reflect.get(log.rows, k); })
    .sort(function (a, b) { return (a.lastSeen < b.lastSeen) ? 1 : -1; });
}

// ---- auto-capture (compatible with the official initAutoErrorCapture) ---
// If weld.diag is present it already hooks console/window/promise errors,
// so we subscribe to its stream instead of double-hooking. Otherwise we
// install our own minimal hooks. Either way feeds the de-duplicated log.
function initAutoErrorCapture() {
  if (meta.capturing) return;
  meta.capturing = true;

  var d = diag();
  if (d && typeof d.subscribe === 'function') {
    // reuse diag's capture; pull its existing entries first
    if (typeof d.entries === 'function') {
      d.entries().forEach(function (e) {
        if (e.level === 'error' || e.level === 'warn') logError(e.source || 'diag', e.message, e.data);
      });
    }
    d.subscribe(function (e) {
      if (e && (e.level === 'error' || e.level === 'warn')) logError(e.source || 'diag', e.message, e.data);
    });
    meta.via = 'diag';
    return;
  }

  // standalone hooks (mirror the official plugin's listeners)
  try {
    window.addEventListener('error', function (e) {
      var msg = (e && e.message) || 'unknown error';
      logError('window', msg, { filename: e && e.filename, line: e && e.lineno, col: e && e.colno, stack: e && e.error && e.error.stack && String(e.error.stack) });
    });
    window.addEventListener('unhandledrejection', function (e) {
      var r = e && e.reason;
      logError('promise', 'Unhandled rejection: ' + ((r && r.message) || safeStr(r)), { stack: r && r.stack && String(r.stack) });
    });
    if (console && typeof console.error === 'function' && !console.error.__weldReportHooked) {
      var orig = console.error.bind(console);
      var hooked = function () {
        try { logError('console', Array.prototype.map.call(arguments, safeStr).join(' '), null); } catch (e) {}
        return orig.apply(console, arguments);
      };
      hooked.__weldReportHooked = true;
      console.error = hooked;
    }
  } catch (e) { /* sandbox without window listeners */ }
  meta.via = 'self';
}

// ---- environment ----------------------------------------------------
// Prefer weld.diag.bundle (richer, already collected); else gather a
// minimal equivalent. We add the browser/device fields the official
// plugin collects via ua-parser, best-effort from navigator.
function environment(extra) {
  var d = diag();
  if (d && typeof d.bundle === 'function') {
    try { return d.bundle({ extra: extra }); } catch (e) {}
  }
  var b = {
    generatedAt: nowIso(),
    generator: (typeof generatorName !== 'undefined' ? generatorName : 'unknown'),
    url: (typeof location !== 'undefined' ? location.href : ''),
    userAgent: (typeof navigator !== 'undefined' ? navigator.userAgent : ''),
    platform: (typeof navigator !== 'undefined' ? navigator.platform : ''),
    language: (typeof navigator !== 'undefined' ? navigator.language : ''),
    viewport: (typeof window !== 'undefined' ? { w: window.innerWidth, h: window.innerHeight } : null)
  };
  if (extra) b.extra = extra;
  return b;
}

// ---- platform fingerprint ------------------------------------------
// A cheap, synchronous fingerprint of platform state if weld.monitor has
// already run; we never trigger a full ~100-assertion run from here.
function platformFingerprint() {
  var m = monitor();
  if (!m) return null;
  try {
    if (typeof m.fingerprint === 'function' && m.groups) {
      // monitor.fingerprint expects the tests array; if a prior run cached
      // a report, surface its fingerprint, else note that no run has happened.
      if (meta.lastMonitorReport && meta.lastMonitorReport.fingerprint) return meta.lastMonitorReport.fingerprint;
    }
  } catch (e) {}
  return null;
}
// Let an app hand us a monitor report so its fingerprint rides along.
function attachMonitorReport(report) { if (report) meta.lastMonitorReport = report; }

// ---- the report object ----------------------------------------------
function buildReport(customData) {
  var rep = {
    generatedAt: nowIso(),
    environment: environment(),
    errors: errorLog(),
    platformFingerprint: platformFingerprint()
  };
  if (customData) rep.customData = customData;
  return rep;
}

// ---- human-readable summary ----------------------------------------
// A compact digest the app can show the user or log to console -- the part
// the official plugin never produces (it only emits an encrypted blob).
function summary(rep) {
  rep = rep || buildReport();
  var env = rep.environment || {};
  var lines = new Array();
  lines.push('weld.report summary @ ' + rep.generatedAt);
  lines.push('generator: ' + (env.generator || 'unknown'));
  if (env.userAgent) lines.push('browser: ' + env.userAgent);
  if (env.viewport) lines.push('viewport: ' + env.viewport.w + 'x' + env.viewport.h);
  if (rep.platformFingerprint) lines.push('platform fingerprint: ' + rep.platformFingerprint);
  var errs = rep.errors || new Array();
  if (!errs.length) {
    lines.push('errors: none captured');
  } else {
    lines.push('errors (' + errs.length + ' distinct):');
    errs.slice(0, 8).forEach(function (e) {
      lines.push('  ' + SB + e.source + ' x' + e.count + SC + ' ' + e.message + (e.count > 1 ? '  (last ' + e.lastSeen + ')' : ''));
    });
    if (errs.length > 8) lines.push('  ... and ' + (errs.length - 8) + ' more');
  }
  return lines.join('\n');
}

// ---- the official drop-in API --------------------------------------
// createTemporaryDebugInfoUrl(opts): identical signature to the official
// bug-report-plugin. Encrypts the (richer) report with opts.publicKey and
// uploads it as a URL that self-deletes after six months. Encryption is
// MANDATORY -- a publicKey is required, exactly as in the official plugin,
// so a report can never leak the reporter's environment in the clear.
function createTemporaryDebugInfoUrl(opts) {
  opts = opts || {};
  if (!opts.publicKey) {
    return Promise.reject(new Error('weld.report: must provide publicKey for encrypting debug info. This is necessary to increase user privacy (i.e. prevent leaking the user\'s browser version, device model, etc.). Generate one at https://perchance.org/public-key-encryption-tool'));
  }
  var rep = buildReport(opts.customData);
  var payload = safeStr(rep);

  return Promise.resolve().then(function () {
    // preload the platform plugins (no-op if already loaded), matching official
    try { if (typeof dynamicImport === 'function') { dynamicImport('secret-plugin', 'preload'); dynamicImport('upload-plugin', 'preload'); } } catch (e) {}

    if (typeof dynamicImport !== 'function') throw new Error('weld.report: dynamic-import-plugin not available');
    return dynamicImport('secret-plugin', 'async');
  }).then(function (secret) {
    if (!secret || typeof secret.encrypt !== 'function') throw new Error('weld.report: secret-plugin unavailable');
    var encrypted = secret.encrypt(payload, opts.publicKey);
    return dynamicImport('upload-plugin', 'async').then(function (uploadPlugin) {
      if (typeof uploadPlugin !== 'function') throw new Error('weld.report: upload-plugin unavailable');
      // expire after six months, matching the official plugin
      var sixMonths = 1000 * 60 * 60 * 24 * 30 * 6;
      return Promise.resolve(uploadPlugin(encrypted, { expires: Date.now() + sixMonths }));
    });
  }).then(function (res) {
    res = res || {};
    if (res.error) throw new Error('weld.report: upload failed (' + res.error + ')');
    return (res.url != null) ? String(res.url) : null;
  }).catch(function (e) {
    try { console.error(e); } catch (x) {}
    return null;
  });
}

window.weld.report = {
  // official drop-in surface
  createTemporaryDebugInfoUrl: createTemporaryDebugInfoUrl,
  initAutoErrorCapture: initAutoErrorCapture,
  // richer weld surface
  report: buildReport,
  summary: summary,
  errorLog: errorLog,
  logError: logError,
  environment: environment,
  attachMonitorReport: attachMonitorReport,
  attach: attach,
  version: '1.0.0'
};

window.weld.report.attach({
  dynamicImport: (typeof dynamicImport === 'function') ? dynamicImport : null
});

return window.weld.report;
})();
</script>
