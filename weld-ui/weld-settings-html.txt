  <h2>Overview</h2>
<p>weld.settings handles the perennial "preferences panel" problem: define a schema, get a typed object you can read and write, automatic persistence, automatic UI. Define your settings once, and the plugin gives you both a programmatic accessor and a working settings panel UI without you writing any DOM code.</p>
<p>Values persist to <code>localStorage</code> by default. Type validation happens on every <code>set</code> call -- numbers get clamped to their range, selects reject unknown options, booleans get coerced.</p>

<h2>Schema fields</h2>
<p>Each schema entry is an object describing one setting. Supported field types:</p>

<h3>type: 'string'</h3>
<pre data-codeid="code4"></pre>
<p>For sensitive values, add <code>secret: true</code> -- the panel renders a password field.</p>

<h3>type: 'number'</h3>
<pre data-codeid="code5"></pre>
<p>Values are clamped to <code>min</code>/<code>max</code>. <code>step</code> is passed to the input.</p>

<h3>type: 'boolean'</h3>
<pre data-codeid="code6"></pre>
<p>Coerced to true/false. Renders as a checkbox.</p>

<h3>type: 'select'</h3>
<pre data-codeid="code7"></pre>

<h3>type: 'color'</h3>
<pre data-codeid="code8"></pre>
<p>Renders an HTML color picker. Value is a hex string.</p>

<h3>type: 'textarea'</h3>

<h2>Reading and writing</h2><h3>settings.get(key) / .getAll()</h3>
<div class="sig" data-codeid="sig10"></div>
<div class="sig" data-codeid="sig11"></div>
<p>Retrieve a single value or a shallow copy of all values.</p>
<pre data-codeid="code12"></pre>
<h3>settings.set(key, value)</h3>
<div class="sig" data-codeid="sig13"></div>
<p>Set a value. Validates against the schema, coerces if needed (number clamping, boolean coercion), throws for unknown keys or invalid select options. Returns <code>true</code> if the value actually changed; <code>false</code> if the new value equaled the old.</p>
<pre data-codeid="code14"></pre>
<h3>settings.reset(key?)</h3>
<div class="sig" data-codeid="sig15"></div>
<p>Reset one key (if given) or all keys to their schema defaults. Fires <code>subscribe</code> callbacks for each changed value.</p>
<h2>Subscriptions</h2><h3>settings.subscribe(key, fn)</h3>
<div class="sig" data-codeid="sig16"></div>
<div class="sig" data-codeid="sig17"></div>
<p>Listen for changes. Per-key subscribers get <code>(newValue, oldValue)</code>. The wildcard <code>'*'</code> subscriber gets <code>(key, newValue, oldValue)</code>.</p>
<pre data-codeid="code18"></pre>
<h2>Building UI</h2><h3>settings.buildPanel(opts?)</h3>
<div class="sig" data-codeid="sig19"></div>
<p>Returns a complete, mounted-ready settings panel. Without <code>sections</code>, every schema key is rendered in order. With <code>sections</code>, keys are grouped under headers; any keys not assigned to a section are rendered at the bottom.</p>
<pre data-codeid="code20"></pre>
<p>The panel uses CSS variables (<code>--weld-ink</code>, <code>--weld-faint</code>, <code>--weld-border</code>, <code>--weld-muted</code>) so it inherits the surrounding theme. If weld.ui is loaded, those variables are already set by the theme system.</p>
<h3>settings.buildField(key)</h3>
<div class="sig" data-codeid="sig21"></div>
<p>Build just one field. Useful for embedding a setting inside a custom layout rather than a full panel.</p>
<h2>Common patterns</h2><h3>Apply settings as they change</h3>
<pre data-codeid="code22"></pre>
<h3>Persistent state with a custom store</h3>
<p>Pass a custom <code>storage</code> with <code>getItem</code> / <code>setItem</code> to swap out localStorage -- useful for switching to IndexedDB via weld.migrate, or for in-memory state during tests.</p>
<pre data-codeid="code23"></pre>
<h2>Caveats</h2><div class="note">
Storage is JSON-serialized. Schema values that don't round-trip through JSON (functions, class instances, Symbols) won't persist correctly. Stick to primitives, arrays of primitives, and plain objects.

// -----------------------------------------------------------------
// Validation and coercion per type
// -----------------------------------------------------------------
function coerce(s, value) {
  if (s.type === 'number') {
    var n = Number(value);
    if (isNaN(n)) throw new Error('not a number');
    if (s.min != null && n < s.min) n = s.min;
    if (s.max != null && n > s.max) n = s.max;
    return n;
  }
  if (s.type === 'boolean') return !!value;
  if (s.type === 'select') {
    var opts = s.options || [];
    var vals = opts.map(function(o) { return typeof o === 'object' ? o.value : o; });
    if (vals.indexOf(value) === -1) {
      throw new Error('value not in options');
    }
    return value;
  }
  // string, color, textarea
  return value == null ? '' : String(value);
}

// -----------------------------------------------------------------
// Schema instance factory
// -----------------------------------------------------------------
function createSettings(opts) {
  opts = opts || {};
  var schema = opts.schema || {};
  var storageKey = opts.storageKey || 'weld-settings';
  var storage = opts.storage;
  if (storage === undefined) {
    storage = (typeof localStorage !== 'undefined') ? localStorage : null;
  }

  var values = {};
  var listeners = {};
  listeners['*'] = [];

  // Seed defaults
  for (var key in schema) {
    var def = schema[key]['default'];
    values[key] = def !== undefined ? def : null;
  }

  // Load persisted
  if (storage) {
    try {
      var raw = storage.getItem(storageKey);
      if (raw) {
        var loaded = JSON.parse(raw);
        for (var k in loaded) {
          if (schema[k]) values[k] = loaded[k];
        }
      }
    } catch (e) { /* swallow */ }
  }

  function save() {
    if (!storage) return;
    try { storage.setItem(storageKey, JSON.stringify(values)); }
    catch (e) { /* swallow */ }
  }

  function get(key) {
    return values[key];
  }

  function getAll() {
    var out = {};
    for (var k in values) out[k] = values[k];
    return out;
  }

  function set(key, value) {
    if (!schema[key]) throw new Error('unknown setting: ' + key);
    value = coerce(schema[key], value);
    var old = values[key];
    if (old === value) return false;
    values[key] = value;
    save();
    fireListeners(key, value, old);
    return true;
  }

  function fireListeners(key, value, old) {
    var L = listeners[key];
    if (L) {
      for (var i = 0; i < L.length; i++) {
        try { L[i](value, old); } catch (e) { /* swallow */ }
      }
    }
    var Lall = listeners['*'];
    for (var j = 0; j < Lall.length; j++) {
      try { Lall[j](key, value, old); } catch (e) { /* swallow */ }
    }
  }

  function reset(key) {
    if (key) {
      if (schema[key]) set(key, schema[key]['default']);
    } else {
      for (var k in schema) {
        var d = schema[k]['default'];
        if (d !== undefined) set(k, d);
      }
    }
  }

  function subscribe(key, fn) {
    if (typeof fn !== 'function') throw new Error('subscribe requires a function');
    if (!listeners[key]) listeners[key] = [];
    listeners[key].push(fn);
    return function() {
      var idx = listeners[key].indexOf(fn);
      if (idx !== -1) listeners[key].splice(idx, 1);
    };
  }

  function buildField(key) {
    var s = schema[key];
    if (!s) return null;

    var row = document.createElement('div');
    row.className = 'weld-settings-row';

    var label = document.createElement('label');
    label.textContent = s.label || key;
    row.appendChild(label);

    var control;
    if (s.type === 'boolean') {
      control = document.createElement('input');
      control.type = 'checkbox';
      control.checked = !!values[key];
      control.addEventListener('change', function() { set(key, control.checked); });
    } else if (s.type === 'select') {
      control = document.createElement('select');
      var opts = s.options || [];
      for (var i = 0; i < opts.length; i++) {
        var opt = opts[i];
        var val = typeof opt === 'object' ? opt.value : opt;
        var lbl = typeof opt === 'object' ? opt.label : opt;
        var o = document.createElement('option');
        o.value = val;
        o.textContent = lbl;
        if (values[key] === val) o.selected = true;
        control.appendChild(o);
      }
      control.addEventListener('change', function() { set(key, control.value); });
    } else if (s.type === 'number') {
      control = document.createElement('input');
      control.type = 'number';
      if (s.min != null) control.min = s.min;
      if (s.max != null) control.max = s.max;
      if (s.step != null) control.step = s.step;
      control.value = values[key];
      control.addEventListener('change', function() {
        var v = parseFloat(control.value);
        if (!isNaN(v)) set(key, v);
      });
    } else if (s.type === 'color') {
      control = document.createElement('input');
      control.type = 'color';
      control.value = values[key] || '#000000';
      control.addEventListener('change', function() { set(key, control.value); });
    } else if (s.type === 'textarea') {
      control = document.createElement('textarea');
      control.rows = s.rows || 3;
      control.value = values[key] || '';
      control.addEventListener('change', function() { set(key, control.value); });
    } else {
      control = document.createElement('input');
      control.type = s.secret ? 'password' : 'text';
      control.value = values[key] || '';
      control.addEventListener('change', function() { set(key, control.value); });
    }

    row.appendChild(control);

    if (s.help) {
      var helpEl = document.createElement('div');
      helpEl.className = 'weld-settings-help';
      helpEl.textContent = s.help;
      row.appendChild(helpEl);
    }

    return row;
  }

  function buildPanel(panelOpts) {
    injectCss();
    panelOpts = panelOpts || {};
    var panel = document.createElement('div');
    panel.className = 'weld-settings-panel';

    if (panelOpts.title) {
      var h = document.createElement('h3');
      h.textContent = panelOpts.title;
      panel.appendChild(h);
    }

    var sections = panelOpts.sections;
    var rendered = {};

    if (sections && sections.length) {
      for (var i = 0; i < sections.length; i++) {
        var section = sections[i];
        var sec = document.createElement('div');
        sec.className = 'weld-settings-section';
        if (section.title) {
          var t = document.createElement('h4');
          t.textContent = section.title;
          sec.appendChild(t);
        }
        var keys = section.keys || [];
        for (var j = 0; j < keys.length; j++) {
          var field = buildField(keys[j]);
          if (field) {
            sec.appendChild(field);
            rendered[keys[j]] = true;
          }
        }
        panel.appendChild(sec);
      }
    }

    // Render any keys not yet in a section
    for (var key in schema) {
      if (rendered[key]) continue;
      var f = buildField(key);
      if (f) panel.appendChild(f);
    }

    return panel;
  }

  return {
    get: get,
    set: set,
    getAll: getAll,
    reset: reset,
    subscribe: subscribe,
    buildPanel: buildPanel,
    buildField: buildField,
    schema: schema
  };
}

window.weld.settings = {
  create: createSettings,
  version: '1.0.0'
};

})();

</script>
