  <h2>Overview</h2>
<p>weld.toast covers the three most-needed transient UI patterns: toast notifications (auto-dismissing alerts), modal prompts (async text input), and confirms (async yes/no). All three are Promise-friendly -- await them inline rather than chaining callbacks.</p>
<p>Toasts queue automatically with a configurable max, stack in any corner, and animate in and out. Prompts and confirms render in a centered modal with a backdrop and keyboard shortcuts (Enter to submit, Escape to cancel).</p>

<h2>Toasts</h2><h3>weld.toast.show(message, opts?)</h3>
<div class="sig" data-codeid="sig3"></div>
<p>Show a generic toast. Returns a handle you can pass to <code>dismiss()</code> to close early. Options:</p>
<ul>
<li><code>type</code>: <code>'info'</code>, <code>'success'</code>, <code>'warn'</code>, <code>'error'</code>. Default <code>'info'</code>.</li>
<li><code>duration</code>: milliseconds until auto-dismiss. Default 4000. Pass <code>0</code> for sticky.</li>
<li><code>position</code>: where to render this toast. Overrides the global default.</li>
<li><code>dismissible</code>: show a close button. Default true.</li>
</ul>
<pre data-codeid="code4"></pre>
<h3>Type shortcuts</h3>
<p>Convenience wrappers for the four common types. Same return value as <code>show</code>.</p>
<pre data-codeid="code5"></pre>
<h3>weld.toast.dismiss(handle) / .clear()</h3>
<p><code>dismiss(handle)</code> closes a specific toast early. <code>clear()</code> removes all active toasts immediately.</p>
<h3>weld.toast.configure(opts)</h3>
<div class="sig" data-codeid="sig6"></div>
<p>Set global defaults. Apply once at startup; individual <code>show()</code> calls can override.</p>
<ul>
<li><code>position</code>: <code>'top-right'</code> (default), <code>'top-left'</code>, <code>'top-center'</code>, <code>'bottom-right'</code>, <code>'bottom-left'</code>, <code>'bottom-center'</code>.</li>
<li><code>duration</code>: default auto-dismiss in ms. Default 4000.</li>
<li><code>max</code>: maximum concurrent toasts before old ones start dropping. Default 5.</li>
</ul>
<h2>Prompts</h2><h3>weld.toast.prompt(opts?)</h3>
<div class="sig" data-codeid="sig7"></div>
<p>Show a modal text input. Resolves with the entered string, or <code>null</code> if the user cancels.</p>
<ul>
<li><code>title</code>, <code>message</code>: shown above the input.</li>
<li><code>placeholder</code>, <code>defaultValue</code>: standard input attributes.</li>
<li><code>multiline</code>: render as a textarea instead of an input. Default false.</li>
<li><code>okLabel</code>, <code>cancelLabel</code>: button text. Default 'OK' / 'Cancel'.</li>
</ul>
<pre data-codeid="code8"></pre>
<p>Single-line prompts handle Enter (submit) and Escape (cancel) automatically. Multiline prompts only respond to button clicks so newlines in the textarea aren't accidentally submitted.</p>
<h2>Confirms</h2><h3>weld.toast.confirm(opts?)</h3>
<div class="sig" data-codeid="sig9"></div>
<p>Show a yes/no confirmation. Resolves <code>true</code> if the user clicks the OK button, <code>false</code> for cancel.</p>
<ul>
<li><code>title</code>, <code>message</code>: standard.</li>
<li><code>okLabel</code>, <code>cancelLabel</code>: button text.</li>
<li><code>danger</code>: render the OK button in red. Use for destructive actions. Default false.</li>
</ul>
<pre data-codeid="code10"></pre>
<h2>Theming</h2>
<p>All UI uses CSS variables: <code>--weld-ink</code>, <code>--weld-surface</code>, <code>--weld-faint</code>, <code>--weld-border</code>, <code>--weld-muted</code>. If weld.ui is loaded, the theme system sets these for you and the toasts/modals automatically match your generator's look. Without weld.ui, sensible dark-mode defaults are baked in.</p>

<h2>Caveats</h2><div class="note">
Both <code>prompt</code> and <code>confirm</code> are modal -- they block interaction with the page beneath the backdrop. If you have multiple actions that could race (a setTimeout firing during a prompt, etc.), make sure your code awaits the result before proceeding.

// -----------------------------------------------------------------
// Toast host management
// -----------------------------------------------------------------
var config = {
  position: 'top-right',
  duration: 4000,
  max: 5
};
var hosts = {};
var activeToasts = [];

function getHost(position) {
  if (hosts[position]) return hosts[position];
  var h = document.createElement('div');
  h.className = 'weld-toast-host ' + position;
  document.body.appendChild(h);
  hosts[position] = h;
  return h;
}

function configure(opts) {
  if (!opts) return;
  if (opts.position) config.position = opts.position;
  if (opts.duration != null) config.duration = opts.duration;
  if (opts.max != null) config.max = opts.max;
}

// -----------------------------------------------------------------
// Toast notification
// -----------------------------------------------------------------
var ICONS = {
  info:    'i',
  success: '\u2713',
  warn:    '!',
  error:   'x'
};

function show(message, opts) {
  injectCss();
  opts = opts || {};
  var type = opts.type || 'info';
  var duration = opts.duration != null ? opts.duration : config.duration;
  var position = opts.position || config.position;
  var dismissible = opts.dismissible !== false;

  var host = getHost(position);

  var toast = document.createElement('div');
  toast.className = 'weld-toast ' + type;

  var icon = document.createElement('div');
  icon.className = 'weld-toast-icon';
  icon.textContent = ICONS[type] || ICONS.info;
  toast.appendChild(icon);

  var msg = document.createElement('div');
  msg.className = 'weld-toast-msg';
  msg.textContent = String(message);
  toast.appendChild(msg);

  if (dismissible) {
    var close = document.createElement('button');
    close.className = 'weld-toast-close';
    close.textContent = '\u00d7';
    close.setAttribute('aria-label', 'close');
    close.addEventListener('click', function() { dismiss(handle); });
    toast.appendChild(close);
  }

  // Enforce max
  while (activeToasts.length >= config.max) {
    var oldest = activeToasts.shift();
    if (oldest) dismiss(oldest, true);
  }

  host.appendChild(toast);
  // Trigger CSS transition
  setTimeout(function() { toast.classList.add('shown'); }, 10);

  var timer = null;
  var handle = { toast: toast, host: host, dismissed: false, _t: null };
  activeToasts.push(handle);

  if (duration > 0) {
    handle._t = setTimeout(function() { dismiss(handle); }, duration);
  }

  return handle;
}

function dismiss(handle, immediate) {
  if (!handle || handle.dismissed) return;
  handle.dismissed = true;
  if (handle._t) clearTimeout(handle._t);
  var idx = activeToasts.indexOf(handle);
  if (idx !== -1) activeToasts.splice(idx, 1);

  if (immediate) {
    if (handle.toast.parentNode) handle.toast.parentNode.removeChild(handle.toast);
    return;
  }
  handle.toast.classList.remove('shown');
  setTimeout(function() {
    if (handle.toast.parentNode) handle.toast.parentNode.removeChild(handle.toast);
  }, 240);
}

function clearAll() {
  while (activeToasts.length) {
    dismiss(activeToasts[0], true);
  }
}

function buildBackdrop() {
  injectCss();
  var bg = document.createElement('div');
  bg.className = 'weld-toast-backdrop';
  document.body.appendChild(bg);
  setTimeout(function() { bg.classList.add('shown'); }, 10);
  return bg;
}

function teardown(bg) {
  bg.classList.remove('shown');
  setTimeout(function() {
    if (bg.parentNode) bg.parentNode.removeChild(bg);
  }, 200);
}

function confirmDialog(opts) {
  opts = opts || {};
  var title = opts.title || 'Confirm';
  var message = opts.message || 'Are you sure?';
  var okLabel = opts.okLabel || 'OK';
  var cancelLabel = opts.cancelLabel || 'Cancel';
  var danger = opts.danger === true;

  return new Promise(function(resolve) {
    var bg = buildBackdrop();
    var modal = document.createElement('div');
    modal.className = 'weld-toast-modal';

    var h = document.createElement('h4');
    h.textContent = title;
    modal.appendChild(h);

    var p = document.createElement('p');
    p.textContent = message;
    modal.appendChild(p);

    var actions = document.createElement('div');
    actions.className = 'weld-toast-actions';

    var cancel = document.createElement('button');
    cancel.className = 'weld-toast-btn ghost';
    cancel.textContent = cancelLabel;
    cancel.addEventListener('click', function() { teardown(bg); resolve(false); });
    actions.appendChild(cancel);

    var ok = document.createElement('button');
    ok.className = 'weld-toast-btn ' + (danger ? 'danger' : 'primary');
    ok.textContent = okLabel;
    ok.addEventListener('click', function() { teardown(bg); resolve(true); });
    actions.appendChild(ok);

    modal.appendChild(actions);
    bg.appendChild(modal);
    setTimeout(function() { ok.focus(); }, 100);
  });
}

function promptDialog(opts) {
  opts = opts || {};
  var title = opts.title || 'Prompt';
  var message = opts.message || '';
  var placeholder = opts.placeholder || '';
  var defaultValue = opts.defaultValue || '';
  var multiline = opts.multiline === true;
  var okLabel = opts.okLabel || 'OK';
  var cancelLabel = opts.cancelLabel || 'Cancel';

  return new Promise(function(resolve) {
    var bg = buildBackdrop();
    var modal = document.createElement('div');
    modal.className = 'weld-toast-modal';

    var h = document.createElement('h4');
    h.textContent = title;
    modal.appendChild(h);

    if (message) {
      var p = document.createElement('p');
      p.textContent = message;
      modal.appendChild(p);
    }

    var input;
    if (multiline) {
      input = document.createElement('textarea');
    } else {
      input = document.createElement('input');
      input.type = 'text';
    }
    input.placeholder = placeholder;
    input.value = defaultValue;
    modal.appendChild(input);

    var actions = document.createElement('div');
    actions.className = 'weld-toast-actions';

    var cancel = document.createElement('button');
    cancel.className = 'weld-toast-btn ghost';
    cancel.textContent = cancelLabel;
    cancel.addEventListener('click', function() { teardown(bg); resolve(null); });
    actions.appendChild(cancel);

    var ok = document.createElement('button');
    ok.className = 'weld-toast-btn primary';
    ok.textContent = okLabel;
    ok.addEventListener('click', function() { teardown(bg); resolve(input.value); });
    actions.appendChild(ok);

    modal.appendChild(actions);
    bg.appendChild(modal);
    setTimeout(function() { input.focus(); input.select(); }, 100);

    if (!multiline) {
      input.addEventListener('keydown', function(e) {
        if (e.key === 'Enter') { teardown(bg); resolve(input.value); }
        if (e.key === 'Escape') { teardown(bg); resolve(null); }
      });
    }
  });
}

  show: show,
  info:    function(m, o) { o = o || {}; o.type = 'info';    return show(m, o); },
  success: function(m, o) { o = o || {}; o.type = 'success'; return show(m, o); },
  warn:    function(m, o) { o = o || {}; o.type = 'warn';    return show(m, o); },
  error:   function(m, o) { o = o || {}; o.type = 'error';   return show(m, o); },
  dismiss: dismiss,
  clear:   clearAll,
  configure: configure,
  confirm: confirmDialog,
  prompt:  promptDialog,
  version: '1.0.0'
};
