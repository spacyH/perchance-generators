      <h1>weld.queue</h1>
      <span class="ver">v1.0.0</span>
    </div>
    <p class="lead">Async task pipeline with concurrency control, retry, cancel, pause/resume. The image generation queue pattern, generalized.</p>
  </div>
  <h2>Overview</h2>
<p>weld.queue is the async task pipeline every image-generating Perchance app reinvents. Add tasks, control how many run in parallel, get progress callbacks, retry on failure with exponential backoff, pause/resume, cancel individual tasks or everything, and await full drain.</p>
<p>weld.queue provides the common infrastructure for you wire up the task function and let the runner handle concurrency, retry, and lifecycle.</p>
<h2>Quick start</h2>
<pre data-codeid="code1"></pre>
<pre data-codeid="code2"></pre>
<h2>Configuration</h2>
<p>Options passed to <code>create()</code>:</p>
<ul>
<li><code>concurrency</code>: max parallel tasks. Default 1 (serial). The slideshow uses 1-4 for image generation.</li>
<li><code>retries</code>: how many times to retry a failed task. Default 0 (no retry).</li>
<li><code>retryDelay</code>: ms before the first retry. Default 1000.</li>
<li><code>retryBackoff</code>: multiplier for each subsequent retry. Default 2 (exponential).</li>
</ul>
<h2>Adding tasks</h2>
<div class="sig" data-codeid="sig3"></div>
<div class="sig" data-codeid="sig4"></div>
<p>The task function <code>fn(data, task)</code> receives its data argument and the task object itself. It should return a Promise (or a value). The task object has <code>id</code>, <code>status</code> (<code>pending</code>/<code>active</code>/<code>done</code>/<code>failed</code>/<code>cancelled</code>), <code>result</code>, <code>error</code>.</p>
<p><code>addMany</code> adds one task per element in the data array, all using the same function. Returns the array of task objects.</p>
<pre data-codeid="code5"></pre>
<h2>Events</h2>
<div class="sig" data-codeid="sig6"></div>
<p>Four events:</p>
<ul>
<li><code>progress</code>: fires after every state change. Receives <code>\{pending, active, completed, failed, total\}</code>.</li>
<li><code>done</code>: fires when a single task completes. Receives <code>\{task, value\}</code>.</li>
<li><code>error</code>: fires when a task exhausts all retries. Receives <code>\{task, error\}</code>.</li>
<li><code>drain</code>: fires when pending + active both hit zero. Receives the final stats object.</li>
</ul>
<h2>Control</h2>
<div class="sig" data-codeid="sig7"></div>
<div class="sig" data-codeid="sig8"></div>
<div class="sig" data-codeid="sig9"></div>
<div class="sig" data-codeid="sig10"></div>
<div class="sig" data-codeid="sig11"></div>
<p><code>cancel</code> removes a pending task or marks an active one for cancellation (it won't retry). <code>pause</code> stops launching new tasks; active ones finish. <code>resume</code> restarts the runner. <code>setConcurrency</code> adjusts on the fly -- if you increase it while paused then resume, the runner fills the new slots immediately.</p>
<h2>Waiting for completion</h2>
<div class="sig" data-codeid="sig12"></div>
<p>Returns a Promise that resolves when all pending and active tasks have finished (or failed). If the queue is already empty, resolves immediately. Use this for batch operations where you need to wait for everything:</p>
<pre data-codeid="code13"></pre>
<h2>Stats</h2>
<div class="sig" data-codeid="sig14"></div>
<p>Snapshot of the current queue state. Call anytime -- doesn't mutate anything.</p>
<h2>Retry behavior</h2><div class="note">
Tasks that throw or reject are retried <code>retries</code> times with exponential backoff before being marked as failed. A cancelled task is never retried. Active tasks can't be force-killed (Promises don't support that) -- cancellation just prevents the next retry.

<script>
(function(){
'use strict';

if (!window.weld) window.weld = {};
if (window.weld.queue) return; // already loaded

// -----------------------------------------------------------------
// weld.queue: Async task pipeline.
// Configurable concurrency (1-N), progress callbacks, retry with
// backoff, cancel, pause/resume. The slideshow image-gen queue
// pattern, generalized.
// -----------------------------------------------------------------

function createQueue(opts) {
  opts = opts || {};
  var maxConcurrent = Math.max(1, opts.concurrency || 1);
  var retryCount = opts.retries || 0;
  var retryDelay = opts.retryDelay || 1000;
  var retryBackoff = opts.retryBackoff || 2;

  var pending = [];
  var active = [];
  var completed = [];
  var failed = [];
  var paused = false;
  var listeners = { progress: [], done: [], error: [], drain: [] };

  function emit(event, data) {
    var fns = listeners[event] || [];
    for (var i = 0; i < fns.length; i++) {
      try { fns[i](data); } catch (e) { /* swallow */ }
    }
  }

  function emitProgress() {
    emit('progress', {
      pending: pending.length,
      active: active.length,
      completed: completed.length,
      failed: failed.length,
      total: pending.length + active.length + completed.length + failed.length
    });
  }

  function runNext() {
    if (paused) return;
    while (active.length < maxConcurrent && pending.length > 0) {
      var task = pending.shift();
      active.push(task);
      emitProgress();
      executeTask(task);
    }
  }

  function executeTask(task) {
    var attempt = 0;
    function tryOnce() {
      var result;
      try {
        result = Promise.resolve(task.fn(task.data, task));
      } catch (e) {
        result = Promise.reject(e);
      }
      result.then(function(value) {
        var idx = active.indexOf(task);
        if (idx >= 0) active.splice(idx, 1);
        task.result = value;
        task.status = 'done';
        completed.push(task);
        emit('done', { task: task, value: value });
        emitProgress();
        if (pending.length === 0 && active.length === 0) emit('drain', stats());
        else runNext();
      }).catch(function(err) {
        attempt++;
        if (attempt <= retryCount && !task.cancelled) {
          var delay = retryDelay * Math.pow(retryBackoff, attempt - 1);
          setTimeout(tryOnce, delay);
        } else {
          var idx = active.indexOf(task);
          if (idx >= 0) active.splice(idx, 1);
          task.error = err;
          task.status = 'failed';
          failed.push(task);
          emit('error', { task: task, error: err });
          emitProgress();
          if (pending.length === 0 && active.length === 0) emit('drain', stats());
          else runNext();
        }
      });
    }
    task.status = 'active';
    tryOnce();
  }

  // -------- PUBLIC --------
  function add(fn, data) {
    var task = {
      id: 't_' + Date.now().toString(36) + Math.random().toString(36).slice(2, 6),
      fn: fn,
      data: data || null,
      status: 'pending',
      result: null,
      error: null,
      cancelled: false,
      addedAt: Date.now()
    };
    pending.push(task);
    emitProgress();
    runNext();
    return task;
  }

  function addMany(fn, dataArray) {
    var tasks = [];
    for (var i = 0; i < dataArray.length; i++) {
      tasks.push(add(fn, dataArray[i]));
    }
    return tasks;
  }

  function cancel(taskOrId) {
    var id = typeof taskOrId === 'string' ? taskOrId : taskOrId.id;
    // Remove from pending
    for (var i = pending.length - 1; i >= 0; i--) {
      if (pending[i].id === id) {
        pending[i].status = 'cancelled';
        pending[i].cancelled = true;
        pending.splice(i, 1);
        emitProgress();
        return true;
      }
    }
    // Mark active as cancelled (will reject on next attempt)
    for (var j = 0; j < active.length; j++) {
      if (active[j].id === id) {
        active[j].cancelled = true;
        return true;
      }
    }
    return false;
  }

  function cancelAll() {
    while (pending.length) {
      var t = pending.pop();
      t.status = 'cancelled';
      t.cancelled = true;
    }
    for (var i = 0; i < active.length; i++) active[i].cancelled = true;
    emitProgress();
  }

  function pause() {
    paused = true;
  }

  function resume() {
    paused = false;
    runNext();
  }

  function clear() {
    cancelAll();
    active = [];
    pending = [];
    completed = [];
    failed = [];
    emitProgress();
  }

  function stats() {
    return {
      pending: pending.length,
      active: active.length,
      completed: completed.length,
      failed: failed.length,
      total: pending.length + active.length + completed.length + failed.length,
      paused: paused,
      concurrency: maxConcurrent
    };
  }

  function setConcurrency(n) {
    maxConcurrent = Math.max(1, n);
    if (!paused) runNext();
  }

  function on(event, fn) {
    if (!listeners[event]) listeners[event] = [];
    listeners[event].push(fn);
    return function() {
      var arr = listeners[event];
      var idx = arr.indexOf(fn);
      if (idx >= 0) arr.splice(idx, 1);
    };
  }

  // Wait for all currently pending + active tasks to finish
  function drain() {
    if (pending.length === 0 && active.length === 0) {
      return Promise.resolve(stats());
    }
    return new Promise(function(resolve) {
      var unsub = on('drain', function(s) {
        unsub();
        resolve(s);
      });
    });
  }

  return {
    add: add,
    addMany: addMany,
    cancel: cancel,
    cancelAll: cancelAll,
    pause: pause,
    resume: resume,
    clear: clear,
    stats: stats,
    setConcurrency: setConcurrency,
    on: on,
    drain: drain,
    isPaused: function() { return paused; }
  };
}

window.weld.queue = {
  create: createQueue,
  version: '1.0.0'
};

})();

</script>
