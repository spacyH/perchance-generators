      <h1>weld.history</h1>
      <span class="ver">v1.0.0</span>
    </div>
    <p class="lead">Chat session management with auto-save and branching. Create, load, list, rename, delete conversations. The "continue where you left off" layer backed by weld.persist.</p>
  </div>
  <h2>Overview</h2>
<p>weld.history is the session layer for AI chat apps. Create, load, list, rename, delete, and search conversations. Each session wraps a weld.branching tree (if loaded) plus metadata — title, character info, timestamps, message count. Messages are persisted automatically via weld.persist with configurable debounce.</p>
<p>This is the "continue where you left off" infrastructure: on page load, list sessions, let the user pick one, load it, and the full conversation tree is restored in memory.</p>
<p>Hard dependency: weld.persist (required). Soft dependency: weld.branching (recommended; without it, sessions store plain message arrays instead of trees).</p>
<h2>Quick start</h2>
<pre data-codeid="code1"></pre>
<pre data-codeid="code2"></pre>
<h2>Creating a session</h2>
<div class="sig" data-codeid="sig3"></div>
<p>Creates a new session and sets it as active. Optional metadata:</p>
<ul>
<li><code>title</code>: display name. Default <code>'New conversation'</code>.</li>
<li><code>characterId</code> / <code>characterName</code>: for filtering and display.</li>
<li><code>metadata</code>: arbitrary extra data.</li>
</ul>
<p>If weld.branching is loaded, a fresh branching tree is created for the session. Otherwise, messages are stored as a plain array.</p>
<h2>Messages</h2><h3>hist.append(message)</h3>
<div class="sig" data-codeid="sig4"></div>
<p>Append a message to the active session. The message object can have any shape — typically <code>role</code> + <code>content</code>. If weld.branching is loaded, the message becomes a tree node (supporting regeneration and editing). Auto-save triggers after the debounce period (default 2s).</p>
<h3>hist.getMessages() / .getTree()</h3>
<div class="sig" data-codeid="sig5"></div>
<div class="sig" data-codeid="sig6"></div>
<p><code>getMessages()</code> returns the active branch as a flat array (from the branching tree's <code>activeBranch()</code>, or the plain array fallback). <code>getTree()</code> returns the live weld.branching instance for direct tree operations (branch, switchSibling, etc.).</p>
<h2>Session management</h2><h3>hist.list(opts?)</h3>
<div class="sig" data-codeid="sig7"></div>
<p>List sessions sorted by most-recently-updated. Filter by <code>characterId</code>, search by title/character name, limit the result count.</p>
<pre data-codeid="code8"></pre>
<h3>hist.load(sessionId) / .save()</h3>
<div class="sig" data-codeid="sig9"></div>
<div class="sig" data-codeid="sig10"></div>
<p><code>load</code> restores a session from storage: deserializes the branching tree (or message array), sets it as active, and updates the view timestamp. <code>save</code> is called automatically by auto-save but can be called manually for immediate persistence.</p>
<h3>hist.rename() / .getMeta() / .updateMeta() / .delete() / .clear()</h3>
<div class="sig" data-codeid="sig11"></div>
<div class="sig" data-codeid="sig12"></div>
<div class="sig" data-codeid="sig13"></div>
<div class="sig" data-codeid="sig14"></div>
<div class="sig" data-codeid="sig15"></div>
<p>Standard session metadata operations. <code>delete</code> removes both the session metadata and its stored tree. <code>clear</code> removes everything.</p>
<h2>Events</h2>
<div class="sig" data-codeid="sig16"></div>
<p>Listen for session events: <code>'create'</code>, <code>'load'</code>, <code>'save'</code>, <code>'append'</code>, <code>'rename'</code>, <code>'delete'</code>, <code>'clear'</code>. The callback receives <code>(event, data)</code>.</p>
<pre data-codeid="code17"></pre>
<h2>Auto-save</h2>
<p>By default, every <code>append()</code> schedules a save after a 2-second debounce. Configure via <code>create()</code> options:</p>
<pre data-codeid="code18"></pre>
<p>Disable auto-save with <code>autoSave: false</code> and call <code>hist.save()</code> manually.</p>
<h2>Storage structure</h2><div class="note">
weld.history delegates all persistence to weld.persist. The session metadata lives in the <code>'sessions'</code> collection and the serialized trees in the <code>'trees'</code> collection. To export a user's entire chat history, call <code>weld.persist.open('weld-history').exportAll()</code> and pipe the result through weld.backup.

<script>
(function(){
'use strict';

if (!window.weld) window.weld = {};
if (window.weld.history) return; // already loaded

// -----------------------------------------------------------------
// weld.history: Chat session manager.
// Each session = metadata (title, character, timestamps, message count)
//              + a serialized weld.branching tree.
// Backed by weld.persist for storage. Uses weld.branching for in-memory
// tree operations.
//
// Soft dependencies:
//   weld.persist  - required for persistent storage (fatal without it)
//   weld.branching - optional; if not loaded, sessions store raw message
//                    arrays instead of branching trees
// -----------------------------------------------------------------

function uid() {
  return 's_' + Date.now().toString(36) + Math.random().toString(36).slice(2, 8);
}

function createHistory(opts) {
  opts = opts || {};
  var dbName = opts.dbName || 'weld-history';
  var autoSave = opts.autoSave !== false;
  var autoSaveDebounce = opts.autoSaveDebounceMs || 2000;

  // Require weld.persist
  if (!window.weld || !window.weld.persist) {
    throw new Error('weld.history requires weld.persist to be loaded first.');
  }

  var db = window.weld.persist.open(dbName);
  var sessions = db.collection('sessions');
  var trees = db.collection('trees');

  var activeSessionId = null;
  var activeTree = null;        // weld.branching instance if available
  var activeMessages = null;    // fallback: plain array of messages
  var saveTimer = null;
  var changeListeners = [];

  function hasBranching() {
    return !!(window.weld && window.weld.branching);
  }

  function emit(event, data) {
    for (var i = 0; i < changeListeners.length; i++) {
      try { changeListeners[i](event, data); } catch (e) { /* swallow */ }
    }
  }

  function scheduleSave() {
    if (!autoSave || !activeSessionId) return;
    if (saveTimer) clearTimeout(saveTimer);
    saveTimer = setTimeout(function() {
      saveTimer = null;
      save().catch(function(e) {
        console.error('[weld.history] auto-save failed:', e);
      });
    }, autoSaveDebounce);
  }

  // -------- Create a new session --------
  function create(meta) {
    meta = meta || {};
    var id = meta.id || uid();
    var now = Date.now();
    var session = {
      id: id,
      title: meta.title || 'New conversation',
      characterId: meta.characterId || null,
      characterName: meta.characterName || null,
      createdAt: now,
      updatedAt: now,
      messageCount: 0,
      metadata: meta.metadata || null
    };

    // Initialize tree or message array
    if (hasBranching()) {
      activeTree = window.weld.branching.create();
      activeMessages = null;
    } else {
      activeTree = null;
      activeMessages = [];
    }
    activeSessionId = id;

    return sessions.put(session).then(function() {
      return saveTree();
    }).then(function() {
      emit('create', { sessionId: id });
      return id;
    });
  }

  // -------- Load a session --------
  function load(sessionId) {
    activeSessionId = null;
    activeTree = null;
    activeMessages = null;

    return sessions.get(sessionId).then(function(session) {
      if (!session) throw new Error('weld.history: session not found: ' + sessionId);
      return trees.get(sessionId).then(function(treeData) {
        activeSessionId = sessionId;
        if (treeData && treeData.tree && hasBranching()) {
          activeTree = window.weld.branching.create();
          activeTree.deserialize(treeData.tree);
          activeMessages = null;
        } else if (treeData && treeData.messages) {
          activeTree = null;
          activeMessages = treeData.messages;
        } else {
          // Empty session
          if (hasBranching()) {
            activeTree = window.weld.branching.create();
            activeMessages = null;
          } else {
            activeTree = null;
            activeMessages = [];
          }
        }
        // Update view time
        return sessions.update(sessionId, { updatedAt: Date.now() });
      });
    }).then(function() {
      emit('load', { sessionId: sessionId });
      return sessionId;
    });
  }

  // -------- Save the active session --------
  function save() {
    if (!activeSessionId) return Promise.reject(new Error('weld.history: no active session'));
    var treeData = { id: activeSessionId };
    if (activeTree) {
      treeData.tree = activeTree.serialize();
      treeData.messages = null;
    } else {
      treeData.tree = null;
      treeData.messages = activeMessages ? activeMessages.slice() : [];
    }
    var msgCount = activeTree ? activeTree.size() : (activeMessages ? activeMessages.length : 0);
    return Promise.all([
      trees.put(treeData),
      sessions.update(activeSessionId, {
        updatedAt: Date.now(),
        messageCount: msgCount
      })
    ]).then(function() {
      emit('save', { sessionId: activeSessionId });
    });
  }

  function saveTree() {
    var treeData = { id: activeSessionId };
    if (activeTree) {
      treeData.tree = activeTree.serialize();
      treeData.messages = null;
    } else {
      treeData.tree = null;
      treeData.messages = activeMessages ? activeMessages.slice() : [];
    }
    return trees.put(treeData);
  }

  // -------- Append a message to the active session --------
  function append(message) {
    if (!activeSessionId) throw new Error('weld.history: no active session');
    if (activeTree) {
      var id = activeTree.append(message);
      scheduleSave();
      emit('append', { sessionId: activeSessionId, nodeId: id });
      return id;
    } else {
      if (!message.id) message.id = uid();
      activeMessages.push(message);
      scheduleSave();
      emit('append', { sessionId: activeSessionId, messageId: message.id });
      return message.id;
    }
  }

  // -------- Get the active conversation --------
  function getMessages() {
    if (!activeSessionId) return [];
    if (activeTree) {
      return activeTree.activeBranch().map(function(n) { return n.data; });
    }
    return activeMessages ? activeMessages.slice() : [];
  }

  function getTree() {
    return activeTree;
  }

  // -------- List sessions --------
  function list(listOpts) {
    listOpts = listOpts || {};
    return sessions.getAll().then(function(all) {
      // Filter
      if (listOpts.characterId) {
        all = all.filter(function(s) { return s.characterId === listOpts.characterId; });
      }
      if (listOpts.search) {
        var q = String(listOpts.search).toLowerCase();
        all = all.filter(function(s) {
          var t = s.title ? s.title.toLowerCase() : '';
          var cn = s.characterName ? s.characterName.toLowerCase() : '';
          return t.indexOf(q) !== -1 || cn.indexOf(q) !== -1;
        });
      }
      // Sort by updatedAt descending (most recent first)
      all.sort(function(a, b) { return (b.updatedAt || 0) - (a.updatedAt || 0); });
      // Limit
      if (listOpts.limit && listOpts.limit > 0) {
        all = all.slice(0, listOpts.limit);
      }
      return all;
    });
  }

  // -------- Session metadata operations --------
  function rename(sessionId, title) {
    return sessions.update(sessionId, { title: String(title), updatedAt: Date.now() }).then(function() {
      emit('rename', { sessionId: sessionId, title: title });
    });
  }

  function getMeta(sessionId) {
    return sessions.get(sessionId || activeSessionId);
  }

  function updateMeta(sessionId, patch) {
    return sessions.update(sessionId, Object.assign({}, patch, { updatedAt: Date.now() }));
  }

  // -------- Delete --------
  function del(sessionId) {
    if (sessionId === activeSessionId) {
      activeSessionId = null;
      activeTree = null;
      activeMessages = null;
    }
    return Promise.all([
      sessions.delete(sessionId),
      trees.delete(sessionId)
    ]).then(function() {
      emit('delete', { sessionId: sessionId });
      return true;
    });
  }

  function clear() {
    activeSessionId = null;
    activeTree = null;
    activeMessages = null;
    return Promise.all([
      sessions.clear(),
      trees.clear()
    ]).then(function() {
      emit('clear', {});
    });
  }

  // -------- Listeners --------
  function onChange(fn) {
    changeListeners.push(fn);
    return function() {
      var idx = changeListeners.indexOf(fn);
      if (idx >= 0) changeListeners.splice(idx, 1);
    };
  }

  function getActiveId() {
    return activeSessionId;
  }

  function isActive() {
    return activeSessionId !== null;
  }

  return {
    create: create,
    load: load,
    save: save,
    append: append,
    getMessages: getMessages,
    getTree: getTree,
    list: list,
    rename: rename,
    getMeta: getMeta,
    updateMeta: updateMeta,
    delete: del,
    clear: clear,
    onChange: onChange,
    getActiveId: getActiveId,
    isActive: isActive
  };
}

window.weld.history = {
  create: createHistory,
  version: '1.0.0'
};

})();

</script>
